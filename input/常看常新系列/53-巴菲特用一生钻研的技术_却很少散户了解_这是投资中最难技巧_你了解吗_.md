## 精修内容

就在最近，市场上发生了一件很有意思的事情。星巴克（Starbucks）突然宣布，将撤掉现任首席执行官（CEO）拉克斯曼·纳拉辛汉（Laxman Narasimhan），转而聘请餐饮巨头 Chipotle 的 CEO 布莱恩·尼科尔（Brian Niccol）。这一公然挖墙脚的行为，带来了很有趣的股价表现。消息一出，星巴克的股价暴涨了 24%，而对面的 Chipotle 则一度大跌了 13%。这一升一降，相当于是一个价值 300 亿美元的市值转移。为什么一个人的改换门庭，就能够给这两家公司带来如此大的变化呢？具体原因之后会讲，但至少现在我们能够看到一个好的管理层对于一家公司的重要性。

沃伦·巴菲特（Warren Buffett）曾说，好的管理层能让平庸的公司变伟大，而糟糕的管理层则能够毁掉一家好公司。巴菲特的一生都在践行“选股票不如选公司、选公司不如选人”的投资哲学。然而，散户投资者似乎普遍对公司管理层并不感冒。相反，散户还很容易被管理层的各种言论所蛊惑，在投资中迷失自我。

但也必须承认，想要看清公司的管理层，并不是一件容易的事，尤其对于散户来说。可这并不是我们忽略它的理由。相反，由于管理层实在是太重要了，它经常会彻底改变一家公司的投资逻辑，所以我们必须搞懂如何去看清管理层。事实上，多数时候管理层的好坏都是有迹可循的。只要我们遵循这些标准，就能够在多数情况下规避掉那些不可挽回的风险，也能够找到真正值得信赖的公司。

在投资中，公司的管理层很重要，这一点相信不会有人不认可。但是，它对于投资究竟能有多大的影响呢？相信多数人并都并不清楚。

下面这张图是 2000 年到 2013 年微软（Microsoft）的股价表现，至少是有那么点日落西山。那些年的微软，不仅拿手的个人电脑（PC）业务毫无起色，在移动互联网上也是后知后觉，整个公司既臃肿又低效，毫无增长可言。而这张图则是 2014 年到现在微软的股价表现，大概翻了 16 倍。为什么会发生如此翻天覆地的变化呢？现在回看我们才知道，其根本的原因就在于 2014 年现任 CEO 萨提亚·纳德拉（Satya Nadella）的走马上任。这位印度裔技术出身的 CEO 原本并没有得到市场多少的期待，但他上任后的表现却出乎了所有人的意料。他先是大胆地把微软最成功的 Windows 业务从主要战略中拿掉，并极具前瞻性地开始大举投入到云计算当中。而微软的 Azure 云也确实在几年后成为了公司最为重要的增长引擎，没有之一。紧接着，他又在企业文化上进行了大刀阔斧的改革。前任 CEO 史蒂夫·鲍尔默（Steve Ballmer）的管理风格非常强硬，他还很喜欢搞内部竞争，这不仅抑制了协作，也抑制了创新。而纳德拉上任后，则立刻废除了前任的“堆栈排名”，打破了公司部门和部门之间的壁垒。他倡导一种更加包容、以人为本的领导方式。这不仅激发了员工的活力，同时也带来了大量的创新，为微软之后的持续增长打下了基础。业务和文化两手抓，相当于硬件和软件同时升级。纳德拉的上任使得微软这艘“破船”又开始乘风破浪了。随后的微软之所以能够在软件即服务（SaaS, Software as a Service）上杀出一条血路，之所以能够在人工智能（AI）时代的大门打开后又占得先机，都和纳德拉建立起来的企业文化和公司战略密切相关。好的管理层对于投资者来说有多重要？在微软身上，就是零和一的区别。

相反，如果一个管理层特别糟糕，那么再好的一手牌都可能会被打得稀烂。中国的百度（Baidu）就是一个很好的例子。百度可以说是有着无与伦比的外部条件：它的竞争对手谷歌（Google）被赶出了中国。过去的二十多年，百度可以说是独占了中国搜索这一大块市场。但结果如何呢？百度逐渐在中国科技巨头的竞争中败下阵来。这背后的原因也在于管理层。一家能够为了利益纵容骗子医生随意打广告的公司，一家经常性地散播虚假和误导性消息的公司，一家在发布会上竟用外国公司技术谎称自己开发的公司，其管理层的问题不言而喻。这样一家公司，即便它的股价也会因为某些时间段的高光而上涨，但也最终会被打回原形。就像最近无人驾驶汽车“萝卜快跑”（Apollo Go）的大火，不管最终这项技术发展得如何，百度这家公司如果不做管理层上的改革，任何技术上的发展都无法给这家公司带来价值上的提升，而任何股价的上涨恐怕也都会是昙花一现。

从微软和百度这两个例子中我们不难发现，公司管理层对于一家公司的投资价值来说，不仅仅是“重要”这么简单，而是决定性的。好的管理层能够将一家平庸的公司带向辉煌，也能够将一家明星公司打入深渊，这背后是有一定的必然性的。所以说，对于投资者而言，要想真正买到安心投资的公司，了解它的管理层至关重要。那么问题来了：怎么去判断一家公司管理层的好坏呢？

每当我想深入了解这个话题的时候，我发现总有一个名字是绕不开的，那就是巴菲特。巴菲特对于管理层的理解可以说是登峰造极，他也将其和投资做到了融会贯通。接下来，就分享三个巴菲特最为看重的管理层的品质，并结合一些实际的案例来分享一些分析管理层的方法。

第一个巴菲特最看重的管理层品质是诚信。这看似是一个虚无缥缈的品质，但是在公司管理层上，其实是有迹可循的，而且还起到了非常决定性的作用。这里就可以来回答视频开头时遗留下来的那个问题了：星巴克为什么会在换帅后股价大涨呢？这就不得不说说这位被“炒鱿鱼”的前 CEO 拉克斯曼·纳拉辛汉了。纳拉辛汉是在 2023 年下半年上任的，他在上任后就开始给星巴克的咖啡涨价。然而，此举不仅没有给公司带来提升，反而从他上任后的第一次财报季开始，公司的增长便出现了放缓。而在这之后便一发不可收拾，最近几个季度甚至还直接出现了负增长。这么糟糕的业绩，自然是引发了投资者的质疑。面对质疑，纳拉辛汉在最近的一次财报会议上说，最近门店客流量的下降，主要跟天气不好有关。言外之意就是：“这不关我的事，之后天气好了可能就恢复了。”然而，面对这样的借口，投资者压根就不买账。CNBC 的著名主持人吉姆·克莱默（Jim Cramer）在采访纳拉辛汉的节目中就公然回怼说：“我研究了所有做饮料和快餐的公司，除了你以外，没有人说天气是个问题。难道真的不是因为你的咖啡卖得太贵了吗？”照理说，这时纳拉辛汉直接承认错误也就完事了，但他的回答又一次让人摸不着头脑。他说，这主要是因为那些偶尔来光顾的顾客少了，而对于这些客人来说，我们平时无法和他们正常沟通，也就没办法告诉他们星巴克所提供的价值。什么叫“偶尔光顾”？什么叫“无法正常沟通”？显然，纳拉辛汉再次找了一个不痛不痒的理由来搪塞投资者。这种顾左右而言他的行为，就是典型的不诚信。一个无法直面问题的管理层，一个无法诚实面对质疑的管理层，怎么可能能运营好一家公司呢？于是，不仅投资者对其是忍无可忍，董事会也对其失去了耐心，所以才有了近期临时换帅的决定。事实上，隔壁的麦当劳（McDonald's）最近也遇到了和星巴克同样的问题，而他们的处理就完美体现了一个好的管理层是应该如何应对质疑的。麦当劳的管理层在最新的财报会议中，主动承认最近公司的一些业绩放缓，就和他们涨价涨得太快有关。他们决定要推出 5 美元优惠套餐来赢回这些客户。现在已经有一些第三方数据显示，这个挽回举措有了一定的效果。这种直面问题并给出解决方案的管理层，才是真正能够让我们投资者放心的管理层。为什么管理层的诚信对于投资者来说如此重要呢？巴菲特的一句话一针见血。他说，在公共场合误导他人的 CEO，私底下也会误导自己。是的，如果一个人总是面对公众去找理由，说明他本身就无法正视自己的问题。那在公司运营时，他也很可能习惯性地为自己开脱，面对真正要解决的问题时选择视而不见。所以，如果你看到一个管理层经常顾左右而言他，不去直面问题，你就需要非常小心了。这些问题很可能会在公开采访中暴露，也很可能会在财报会议上暴露，只要我们留心，就一定能够发现。

第二个巴菲特最看重的管理层的品质，是他是否有长期视野。巴菲特曾说，很多管理者都会过于关注短期的股价，而忽视公司长期的发展。所以他和查理·芒格（Charlie Munger）在寻找管理者的时候，会特别重视他们是否有足够长的视野。比如，通用汽车（GM, General Motors）的管理层就是典型的短视代表。在 21 世纪初，为了应对短期非常激烈的竞争，公司开始不惜一切代价地去砍成本。他们不仅削减了研发投入，还将原本引以为傲的产品质量抛之脑后。这就直接导致了几年后公司产品线的全面暴雷，并在 2009 年申请了破产保护。另一家公司富国银行（Wells Fargo）的管理层也是如此。他们为了短期的销售目标，推行了非常激进的销售任务。员工们为了完成目标，就不得不铤而走险，开始了大量未经授权的假账户。事情败露后，富国银行遭到了数十亿美元的罚款，并损失了大量的用户，它的股价也一蹶不振。相反，将长期视野发挥得淋漓尽致的代表，则要当属亚马逊（Amazon）了。亚马逊之所以能够有今天的成就，跟杰夫·贝索斯（Jeff Bezos）近乎执拗地坚持长期视野有很大关系。他早在 1997 年的首次公开募股（IPO）募股信上就说过，公司会致力于长期增长，而不是短期的利益最大化。他也说过，亚马逊的每一个重要决策都会是基于其长期的影响，而不是为了迎合短期的市场期望。而这么多年来，亚马逊也的确是这么做的。多年来，亚马逊始终保持着极低的利润，而且非常热衷于“烧钱”来投资未来。同时，围绕在他们周围的质疑声也从未间断过。然而，贝索斯却从不被这些质疑声所影响，始终坚持着他自己的长期视野。像亚马逊的亚马逊网络服务（AWS, Amazon Web Services）以及后来推出的 Amazon Prime 都是典型的代表。它们都是前期大量烧钱、短期又根本看不到效果的项目，市场也一度对其充满了质疑。但却在贝索斯的力排众议下，业务最终都获得了不俗的成效。像 AWS 和 Prime 这种事情，放到一般的管理层上，中途大概率要坚持不下去，因为这些项目的见效时间太长了，中途要发生的意外也会太多。但是贝索斯则秉持着长期视野坚持了下来，也才有了今天亚马逊的成功。所以说，管理层是否有长期视野，对投资者来说至关重要。如果你看到一名管理层过于关注短期股价的变动，股价一有风吹草动，他就跑出来安抚市场或者做出些短视的调整，那你就需要非常地警惕了。你不会放心把钱交给这样的管理层帮你保管的。好的管理层应该会坚持长期真正正确的决定，不会轻易受到市场的干扰，而时间最终会给出我们答案。正如巴菲特所言：“时间是优秀企业的朋友，却是平庸企业的敌人。”说的就是这个道理。

第三个巴菲特最看重的管理层的品质，是他是否对股东负责。巴菲特曾说，管理能力差劲的管理者，常常不会考虑到股东的利益。所以你看，巴菲特投资的公司往往都非常愿意分红或者是股票回购，这些都是典型的回馈股东的方式。巴菲特这话倒不是说只有分红的管理层才是好的管理层，他这话更深层的含义是：当公司看不到有足够高回报的项目时，它是否会把股东利益放到第一位，而不是去做愚蠢的决定，乱投资、乱扩张。这其实并没有那么容易做到，因为很多管理层为了证明自己不那么平庸，常常会强行做些什么，而不是保持克制，简单地去进行分红或者回购。巴菲特在这方面做得就非常好。如果巴菲特看不到好的机会，他宁愿手握大量现金，也不会强行去投资那些自己不认可的公司。虽然巴菲特自己本身不分红，但他能够顶住压力不做蠢事的投资风格，本身也是对于股东利益负责的一种体现。而相反，市面上其实有很多上市公司的管理层根本就做不到这一点。这里就不得不再次拿出纳拉辛汉来“鞭尸”了。纳拉辛汉在星巴克任内，明明已经看到了单店客流量在下滑，他却放任不管，反而还在一意孤行地去投资扩张门店。更让人无法理解的是，纳拉辛汉把最激进的开店计划都放到了中国。要知道，在公司所有已开拓的市场当中，中国的前景是最为暗淡的，也是竞争最为激烈的地区。最近一个季度，星巴克在中国的销售额甚至还大跌了 16%。在这种情况下，眼前的问题都没解决，还在盲目地进行门店扩张，很显然，这就是对股东的不负责任。一名负责任的管理层在这种情况下，应该先解决眼下单店客流量下滑的问题，再去考虑进一步的扩张。不然，扩张出来的门店也逃不开原有的这些麻烦。而在这个过程中，如果公司还有额外的现金流，那给股东分红或者进行股票回购，会远比盲目扩张投资要来得更好。而很显然，纳拉辛汉不甘于这么做，而是总想着要搞些大动作。最终，公司的业绩反而在这种拔苗助长下不断地下滑。所以你看，对股东负责对于管理层来说至关重要。它不只是在回馈股东时的慷慨，更是在公司运营时时刻保持了克制。从股东利益出发，就是在没有好机会的时候，不去做愚蠢的投入；从股东利益出发，就是不被自身的光环所绑架，为做正确的事情去承担全部的质疑。而如果你看到哪个公司的管理层特别地好大喜功，不顾股东利益，盲目地进行投资和扩张，那么你就需要特别谨慎了，因为这往往意味着公司未来很可能将面临不可挽回的风险。

以上就是巴菲特最看重管理层的三大品质：诚信、长期视野和对股东负责。你可以拿着这三点去对照一下你自己的投资，来评估一下它们的管理层是否足够优质。要知道，好的管理层这三点缺一不可，任何一点的缺失都可能会给你的长期投资带来无可挽回的风险。

管理层这一点，一直是我投资中最为关注的风险之一。就像在 20 年初，我曾经就清仓了一只我持有多年的股票——波音（Boeing），而原因正是因为管理层的诚信问题。当时我还特地做了一期视频来详细分享我对于波音的看法。不过后来，波音股价触底反弹了，还有不少人跑到视频下面幸灾乐祸。但在我看来，这样的公司如果管理层不做出改变，它的衰落是必然的。不管中间的股价如何回暖，都无法改变公司长期式微的趋势。现在我们再看波音的表现，无论是它的股价，还是后来不断出现的丑闻，无不验证了我当年的判断。

而这其实也是我今天给大家做这期视频的初衷。在我们的投资中，管理层的优先级往往会放得比较靠后。人们总是喜欢关注股价的变化、关注业绩的表现，或者关注那些长期令人兴奋的故事。但是实际上，一家企业，人才是最为关键的。一个值得信赖的管理层，即便公司遭遇到了困境，你也能相信他有能力将公司带回正轨，你会愿意把钱交给他，让他帮你管理。相反，一个不值得信任的管理层，对于长期价值投资者而言，往往就是一个绝不能容忍的风险。这样的管理层在好的光景下倒也不会出什么太大问题，但一旦公司遭遇困难，情况就会发生彻底的扭转。糟糕的管理层是会放大公司所遇到的一切问题的。不管是宏观上的压力、竞争，抑或是单纯的意外，都会使你的投资风险陡增，因为一不小心，公司可能就跨不过去这道坎了。这样的公司，即便故事再好、业绩再优，也存在付之一炬的风险，而我们在美股市场上已经见过太多这样的案例了。

现在，我们了解了管理层的重要性，也了解了分析管理层的方法，我建议各位都花点时间去给自己的投资做一个评估。你会发现，加入了这层思考之后，很多投资决策可能都会变得不同了。不过也必须承认的是，分析管理层和分析财报不同，它不是拿着一套标准就能立马得出结论的。要想深刻地了解一家公司的管理层，确实需要一定时间的跟踪和积累。但至少，现在我们有了标准也有了方法，未来在看到管理层相应的言论和行为时，你就能提高警惕了。
