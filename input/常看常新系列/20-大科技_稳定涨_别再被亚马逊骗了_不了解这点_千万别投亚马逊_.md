## 精修内容

在美股所有的科技巨头中，亚马逊一直是我认为最难理解，也最难分析的一家公司。我相信许多读者应该或多或少都持有过，或者正在持有亚马逊的股票，但我猜多数人都并不能真正地理解这家公司，因为这家公司实在是存在太多匪夷所思的地方了。

亚马逊的 **CEO** (首席执行官) 是华尔街出身，但他的公司几乎所做的所有事都与华尔街背道而驰。比如，他上市了 20 年，却始终没有盈利；他常年负债累累，却还经常砸钱去做亏钱的项目，一亏就是几十年。若说他有理想，他却没有任何能够拿得出手的、改变世界的项目；若说他是韬光养晦，他又经常大手笔地并购其他公司。有时我都会怀疑，贝佐斯他自己是否知道亚马逊最终要走向何方。

不要觉得这是我目光短浅，其实华尔街的大佬们也经常搞不清楚。在马斯克发迹之前，贝佐斯和他的亚马逊一直都是最让华尔街出乎意料的公司，没有之一。不过，经过了这么多年与亚马逊的死磕，我感觉自己似乎研究出了一条清晰的逻辑线，自此也坚定不移地开始投资这家公司。我不敢说我的研究一定正确，但不妨拿出来与大家详细分析一下，我们各抒己见，说不定还能碰撞出不一样的价值。

在 **FAANG** 这几家大科技公司中，如果我们去看它们早期的样子，会发现亚马逊无疑是处境最艰难的一个。无论是苹果还是谷歌，它们在早期看起来都像是一家前途无量的公司，但亚马逊的早期只能用“挣扎”二字来形容。现在回看它的挣扎，很大程度上是它自找的。

在刚上市的头几年里，亚马逊就是一家连年亏损的公司，而且一年亏得比一年多。亏钱倒不可怕，可怕的是它似乎从来没有想过赚钱这件事。现在的互联网公司也烧钱，但至少都能画出一张关于盈利的大饼来吸引投资者。而在早期的亚马逊商业模型中，似乎根本看不到这张盈利的大饼。那它没钱了该怎么办呢？就是一个字：借。找投资人借，找市场借，然后再次投入到不赚钱的项目中。早期的亚马逊杠杆率就从未低于 50%，而现在最擅长借钱的特斯拉，当初濒临破产时，最高的杠杆率也不过是 38%。

当初亚马逊烧钱烧到什么程度呢？举个例子，当初亚马逊率先推出了无条件退货政策。这个政策刚开始就给本不富裕的亚马逊造成了极大的成本压力。但贝佐斯并不在乎，他认为这能够增加顾客的满意度，所以就不计成本地继续了下去。再比如，**Prime** 刚推出的时候，也被认为是彻头彻尾的赔钱服务。Prime 主打无最低消费、无限制的免运费两日送达。当时的公司 **CFO** (首席财务官) 就警告贝佐斯称，这会导致第一批用户给公司带来巨大成本，产生惊人的快递费用。而结果也确实如此，Prime 使得亚马逊的成本又上了一个台阶。

二十年前，**NBC** 的主持人在采访贝佐斯时就曾调侃他说：“你知道‘利润’这个单词怎么拼写吗？” 确实，这样的一家公司可以说对于投资者来说是非常不友好的。这就是为什么你经常能看到有人持有了十年的苹果、二十年的微软，却很少见到有人持有亚马逊超过十年。这是因为早期的亚马逊很难让人看清其价值所在，而苹果每推出一个产品，你都能看到它是怎么赚钱的，微软和谷歌也是如此。

现在的亚马逊电商业务算是守得云开见月明了，因为它似乎终于找到了利用电商来赚钱的法宝。下面这张图是亚马逊过去三年各个业务的收入数据，前三项都是传统的网上卖货收入。别看这部分收入高，但都是不赚钱的，在这一点上，亚马逊用了 25 年的时间，将赔本赚吆喝进行到底了。它的电商业务真正赚钱的是下面这两项：Prime 订阅服务和广告。

像 Prime 这种订阅服务需要有一定的规模才能够赚钱，而现在的亚马逊在全球已经积累了两亿的 Prime 用户，完全实现了规模化。它现在每增加一个 Prime 用户，几乎不需要任何成本，完全都是利润。而过去几年，亚马逊的 Prime 收入增长也极快，平均增速能够达到 30%。最近亚马逊又宣布 Prime 的年费将上调 20 美元，更是极大地促进了它未来的利润来源。

而亚马逊电商赚钱的另一大法宝——广告，也已经初露锋芒。过去两年，广告业务的平均增速达到了惊人的 57%，这是以前所有投资者都未曾预料到的。虽然亚马逊没有披露这部分的利润率，但参考谷歌和 Facebook 超过 80% 的毛利和 40% 的净利，亚马逊的广告业务也将成为未来的一大块利润来源。值得注意的是，虽然现在 Prime 和广告业务发展得风生水起，但这两项业务真正开始赚钱，也只不过是 2018、2019 年的事情，距离亚马逊上市已经过去了 20 年之久。如果现在有家上市公司说“我们 20 年之后就能赚钱”，我估计肯定没人愿意投资。而亚马逊就是这样一家极其吊诡的公司。

不过别急，更诡异的还在后头。既然亚马逊的电商业务这两年已经找到了盈利的解药，那么它现在是不是就可以毫无顾忌地赚钱了呢？实际上还是不行。细看它的财报你会发现，亚马逊现在的盈利几乎全部都来自于 **AWS** (亚马逊网络服务)，这个我一会会详细分析，我们还是先来看它的电商。它的电商经营利润率仍然只有可怜的 1.5%，要知道，这还是算进了 Prime 和广告这两大赚钱机器后的利润率。而且，虽然 Prime 和广告这两年突飞猛进，但亚马逊电商的利润率却丝毫没有改善。这又是为什么呢？

从下图中我们可以看出，亚马逊的资本支出，也就是购买工厂和设备的钱，在最近两年翻了接近四倍，营收占比从 6% 陡增至 13%。而且这部分钱不是用于最赚钱的 AWS 项目，而是几乎全部投入进了电商业务。更高的资本支出就会带来更高的折旧成本，这部分成本显著压低了电商的利润率。

不过还需要注意的是，这么高的资本支出，光靠 Prime 和广告赚的钱是完全不够用的。那怎么办呢？还是老办法：大举借债。从下图中我们可以看出，亚马逊这家公司从来就没有停止过借债，它的债务增速都快赶上它的股价增速了。尤其在疫情这两年借得更多，长期债务从 630 亿美元陡增至了 1164 亿美元，翻了近一倍。但不要以为这是因为公司规模扩大了好借钱，实际上从它自身的债务占比上看，这个比例也是在不断增加的。

当然了，亚马逊也非常精明，这两年大举借债也跟极低的利率环境有关。下图显示，亚马逊这两年的借贷利率低得吓人，最低时竟然能借到 0.35% 利率的贷款。相比于当时 7.5% 的通胀率来说，这钱简直就是白送的。借钱借到手软的亚马逊当然不能放过这千载难逢的好机会，于是在疫情最为严重之时，它反而逆流而上，开始了又一轮大规模扩张。过去两年，尽管疫情导致招工困难，亚马逊电商的员工却翻了一倍，从 80 万增加到了 160 万。公司还购买了大量的飞机，减少了对于第三方运输的依赖，平均交货时间也从原来的两天缩短到了现在的一天。

亚马逊这种不计成本的投入也确实看到了效果。如今的北美电商市场已经趋于成熟，而且竞争也在日益激烈。在这种情况下，亚马逊的市场份额仍然能够快速增加，从 2016 年的 36% 上升至今年的 43%。垄断地位的提升意味着对供应商的议价能力也在不断增强，这也是零售公司最为看重的一点。亚马逊现在的议价能力可以说在行业内一骑绝尘，甩开传统零售商好几条街。亚马逊的现有库存天数为 38 天，而沃尔玛为 43 天，**Home Depot** 是 69 天，这说明亚马逊的商品周转率极高。这不仅说明亚马逊的货品受欢迎，也意味着卖得不好的货品可以迅速退回给供货商。另一个反映议价能力的数据是应付账款天数，亚马逊为 98 天，沃尔玛和 Home Depot 则分别是 43 天和 42 天。这意味着亚马逊给供货商付款，平均要拖欠三个月，而供货商也无可奈何。

亚马逊的电商业务介绍到这里，不知道各位看官有没有跟我一样的感受，那就是我们似乎和 20 年前一样，又开始看不懂它要做什么了。它明明可以赚钱，却偏要做亏钱的项目；明明摆着一个 AWS 这样的炙手可热的项目，却一直不计成本地把钱投入到电商当中去。它这么无限的投入到底什么时候是个头呢？它真的想过要赚钱吗？对于投资者来说，这样的经营风格又最终会意味着什么呢？我们这里先卖个关子，待我把接下来亚马逊 AWS 的业务也分析完后，很多问题就会迎刃而解了。

AWS 的存在本身就是一件非常吊诡的事情。为什么做软硬结合的老牌科技公司苹果、微软没有做云计算？做软件最强的谷歌也没有做云计算？反而是这么一个靠着电商卖货的公司，最终却最早发展出了云计算呢？要回答这些问题，还要从 AWS 的诞生开始说起。

AWS 最初源于一个点子。有一次，公司的一个技术团队计划用三个月的时间来完成一个项目，可是三个月后，他们只是建立了一些数据库、运算、存储等基础设施，离项目完成还有很远。这时，一名程序员就提出，既然每一次做新项目都要搭建一遍基础设施，那么为什么不能直接搭建一个大的，以后每做一个新项目直接用这个就好了。贝佐斯听到这个点子很是认同，但他的理想远不在此。他意识到，既然可以搭建一个基础设施给全公司用，为什么不能搭建一个基础设施给全世界用呢？于是，他下定决心要进行一次大规模的尝试。

然而，董事会对于这个点子是强烈反对的。他们认为云计算和公司的电商业务毫无关系，而且非常烧钱。要知道，当时的公司还处在严重的亏损当中，这样做无异于火上浇油。不过最终，贝佐斯还是力排众议，开始了一场为期十多年的下注。亚马逊刚开始搞云计算 AWS 的时候，其他科技巨头都不为所动，他们不敢烧这么多钱去做一个完全看不清未来的项目。而等到第一个竞争对手微软终于看清云计算的商机时，AWS 已经做了五年之久。亚马逊也因为贝佐斯的这次疯狂，抓住了巨大的先发优势。

今天的 AWS 是亚马逊毋庸置疑的赚钱引擎，AWS 的利润率达到了 30%，以一己之力撑起了整家公司的利润。下图显示，随着 AWS 营收占比的增加，从 2013 年的 4% 增加到 2021 年的 13%，公司的利润率也从亏损的状态一举跃升至 2021 年的 7%。可以说，亚马逊最近几年的利润提升完全都是 AWS 的功劳。

AWS 不仅赚钱，它的增速也非常的快。在最新的财报中，AWS 依然能保有 40% 的超高增速。而且现在，整个云计算开拓的市场还不到可覆盖面积的 5%，预计到 2030 年，行业规模将会是现在的四到五倍。从估值上看，现在 AWS 的市值大概占亚马逊总市值的一半，约为 8000 亿美元。我们保守估计，如果 AWS 能够在未来五年内保持 25% 到 30% 的增速，那么五年后，它的估值就将会达到 1.5 万亿美元，相当于现在整个亚马逊的市值。换句话说，就是你现在买亚马逊的股票，五年后光 AWS 就能回本，其他的业务相当于白送给你。

现在，我猜大家应该准备好去了解为什么亚马逊身上存在这么多不合理的地方了。实际上，亚马逊不是不想赚钱，也不是没有能力赚钱，而是它会习惯性地为了更长远的目标而刻意忽略短期的业绩。亚马逊在这一点上是真正做到了极致。一般公司的长期愿景都是长期如何去赚钱，而亚马逊把这一点都当做了短期目标。它的长期主义在于把一件事情做到极致。

对于电商业务来说，做到极致就是追求极致的客户满意度，挖掘出极致的电商使用率。为此，它就不惜一切代价地去投入成本，就连怎么赚钱这么重要的事，它都不去多想。我猜 20 年前，贝佐斯自己都不一定知道电商业务未来能够靠卖订阅会员和广告来赚钱。但他就是这样不顾一切地做，然后在做的过程中摸索生存的方向。现在亚马逊的电商业务同样如此，我们会质疑它投入的资本回报率，会好奇它投入的终极目标要怎么赚钱。对于这些问题，现在的投资者没有答案，我猜亚马逊自己也不一定真的清楚，但它们依旧在沿用同样的思路，将电商做到极致。至于怎么赚钱，之后可能就知道了。

对于云计算 AWS 来说也是如此。贝佐斯坚信云计算是能够改变世界的技术，于是他就烧钱去做，将其做到极致。在这个过程中，他没有考虑投入的回报率，也没有考虑 AWS 与电商的协同效应，这些都是所有其他公司和投资者会关注的事情。他在意的是把一件事情做到极致。最终，AWS 确实是赚钱了，还赚了大钱，但这也不过是碰巧而已。我猜即使不赚钱，贝佐斯也将会继续下去，将 AWS 做到最好仍旧会是我们未来看到的方向。

我们现在经常会听到有人质疑亚马逊，说它相对于其他科技巨头来说缺少创新。但当你真正了解了它的经营风格后，你就能理解，亚马逊不会像 Facebook 那样，搞一个 **AI** (人工智能)，再搞一个元宇宙。亚马逊的重心在于把一件事情做到最好，而且是不惜一切代价地做到最好。曾国藩曾经这样形容自己：“结硬寨，打呆仗。” 我认为用来形容贝佐斯和他的亚马逊也非常贴切。只要是他认准的事情，就一定会做到极致。过程中别人如何评述，就让别人去操心吧。贝佐斯在 1997 年的第一封致股东信中就写道：“亚马逊的一切都关乎于长期。在很长的一段时间内，我们将无利可图，这就是我们的策略。” 很显然，他很早就已经想清楚这一点了。

而对于我们投资者而言，如果你是那种追求确定性，希望能看到明确的赚钱通道的投资者，那么亚马逊这家公司可能不适合你。因为他这种带着长期主义心态将事情做到极致的风格，经常让人看不清方向。这也是为什么你经常能看到亚马逊的股价接连两三年横盘，然后突然出现一波暴涨的情况。如果想要投资亚马逊，我认为最重要的一点是和它分享相同的价值观。这就是说，你也愿意相信追求极致能够最终带来好的回报，但这个回报具体是什么，你并不关心。只有这样，你才能拿得住这家充满争议的公司的股票。

当然，我也不是要盲目地吹捧这种长期追求极致的企业文化。必须得承认，这种企业文化存在很大的风险。AWS 是一个成功的案例，但也不得不说，它具有一定的偶然性。如果当初他烧钱烧到一半真的进行不下去了，亚马逊也很可能真的被拖垮。这种成功是很难被复制的。对于现在的亚马逊来说也同样如此，它现在如此不计成本地去发展电商，最终真的能找到一条康庄大道吗？真的能在极致的产品上找到极致的赚钱策略吗？真的能够停止“输血”，自然地去收获吗？这些问题只有时间才能给我们答案。
