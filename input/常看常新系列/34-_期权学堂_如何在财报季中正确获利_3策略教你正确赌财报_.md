### 策略一：单向买入看涨/看跌期权

第一种策略，也是最基础的赌财报策略，就是单向买入看涨期权 (call option) 或者看跌期权 (put option)。这种方式简单粗暴，又称“裸奔流”赌财报。蒙对了可以无限翻倍，蒙错了期权就可能会变废纸。由于期权可以理解为加了杠杆的股票，这种方式最大的优势就是可以花小钱办大事。以苹果股票为例，目前一张一周后到期的苹果看涨期权合约 (call option contract) 大概五六百美元，而100股苹果股票则需要将近4万美元。五六百美元你可能还输得起，但是4万美元对于多数普通投资者来说可能都拿不出来。所以，选择自带杠杆的期权去赌财报，就是一个完美的以小博大的方式。举个例子，如果你预计苹果的财报会超过预期，也就是说你看涨财报后的苹果，那么你可以选择买入苹果的看涨期权。一旦你蒙对了，那动辄就是成倍的增长。然而，要是蒙错了，股票没涨或者跌了，这五六百美元可能就全打了水漂。

这个策略在实际操作中有几点需要注意：
1.  **尽量选择平价期权 (at-the-money option)**，也就是接近当前股价的行权价格 (strike price)，因为这会使你蒙对方向时的收益最大化。至于亏损，你也不用担心，因为一旦你蒙错，不管怎么选，这个期权最后都不值什么钱，所以相对来讲，最大化蒙对时的收益更加重要。
2.  **在股票的选择上需要根据自己的风险承受能力来判断。** 上文提到苹果的平价期权大概五六百美元，对于美投君自己来说，这个潜在的损失是能接受的。然而像特斯拉这样，一张平价期权动辄七八千美元，对于我来说，这个潜在损失就太高了。所以在用这种方式赌财报时，选择适合自己的目标股票十分关键。
3.  **尽量避免选择财报发布后马上到期的期权。** 因为期权到期行权 (exercise) 是个很麻烦的事，不管你是赚了还是亏了都是如此，尤其对于新手来说。所以最好是选择买一周或者一月后到期的期权，然后在财报发布后马上卖掉，赚取差价。

### 策略二：担保看涨期权

另一种赌财报的策略是上期讲过的担保看涨期权 (covered call) 策略，也就是买入股票的同时，再卖出一个高于当前股价的看涨期权。与上面那种预测大涨或大跌的策略不同，这种策略适合小涨、不涨或者小跌的情况，并且可以在这三种情况下都实现盈利。Covered call 策略的本质是，你期望股票会涨，但是不会涨太多，也就是不会涨过你卖出看涨期权的行权价。然而，即便股价涨过了行权价，你也不会亏钱，仅仅是牺牲了一部分上涨潜力，来换取卖出看涨期权所赚得的权利金 (premium)。具体原理上期做过详细讲解，这里就不赘述了。财报季中，covered call 策略在原有优点的基础上又多了一项，那就是此时的期权价格通常会比平时更贵。一个比较浅显的原因是，这时买期权的人更多了，期权的价格自然也更高。因此，此时卖出看涨期权能够获取比平时更多的收益，也就能弥补更多的潜在下跌空间。

这种策略在实际操作中，就需要选择财报后马上到期的看涨期权来卖出。因为你手里持有股票，不怕被行权。相反，你希望尽快卖出股票，并让看涨期权作废，这样卖出期权赚得的权利金就能稳稳地收入囊中，并且你也不用继续担心股票接下来的波动。Covered call 策略比起“裸奔流”策略，优势在于不管是大涨、小涨、不涨或者小跌，你都能赚钱。即便大跌，你也能用权利金去抵消一部分亏损。缺点就是你需要有买入100股正股 (underlying stock) 的资金，而对于像亚马逊、特斯拉这种千元股价的股票来说，我们普通投资者操作起来就有些吃力了。

### 策略三：垂直价差策略

那么，有没有一种既能控制风险，又不需要大量资金的方法呢？这就提到了另一种赌财报的策略：垂直价差策略 (vertical spread)。

Vertical spread 策略不需要引入正股，只通过操作两个期权便可以实现控制风险并降低成本的目的。它本质上就是在“裸奔流”的基础上再穿了点“衣服”，降低了潜在亏损。所以和“裸奔流”相同，首先你需要先蒙一个方向，是涨还是跌。我们以蒙跌为例，在“裸奔流”中预测股价大跌，你的选择是买入看跌期权。在 vertical spread 中，你同样需要先买一个看跌期权，一般也需要设定行权价在当前股价附近。回到看跌期权最基础的逻辑，买入看跌期权就是看跌，股价越跌你越赚钱。如果再往深想一层，这个“看跌”本质上是说，股价从当前价格（比如说100美元）一路跌到0的整个过程你都能赚钱。也就是说，你买入看跌期权交的这个费用，是让你享受股价从100美元跌到0的这个完整过程的。然而我们都知道，股价一夜之间跌到0的情况基本不会发生，能跌个10%就算很高了。这相当于你为整场盛宴付了钱，结果可能只吃了一道开胃菜就结束了。那怎么办呢？既然大概率只能享用到一部分，那你就只付这一部分的钱不就好了吗？

所以在 vertical spread 中，你是先买了一个看跌期权，这时你可以通过再卖一张相同到期日、但是行权价格更低的看跌期权来实现降低成本的目的。举个例子，比如你可以先买一张行权价100美元的看跌期权合约，假设花了500美元；再卖一张相同到期日、行权价在90美元的看跌期权合约，假设收到200美元的权利金。这时你整个组合的成本就是300美元。这样操作，你的预期是股价会从100美元往下跌，但是跌到90美元就差不多了。如果股价最终没有跌破90美元，那么你赚的钱就跟你“裸奔流”单买看跌期权相同，但是成本却从500美元降到了300美元。如果跌破了90美元，你最多也只能赚到股价跌至90美元为止的利润。当然，如果股价涨了，说明你赌错方向了，你自然要亏钱，不过最多也就亏损300美元。这相当于你牺牲了股价在90美元以下的赚钱机会（一个小概率事件），来换取固定的200美元成本的降低。这种看跌的 vertical spread 叫做熊市看跌垂直价差 (bear put vertical)。相反，如果你看涨，同样可以做 vertical spread，原理相同：你买一张接近当前股价的看涨期权，然后再卖一张行权价更高的看涨期权。这时你的预期便是，股价会涨，但不会涨过你卖出看涨期权的行权价。这种看涨的 vertical spread 就叫做牛市看涨垂直价差 (bull call vertical)。

这么做的好处是，你通过卖出一张期权 (sell option) 所赚得的权利金，降低了整个策略的成本，而整个策略的成本就是你赌财报最多可能亏损的钱，相当于降低了潜在亏损。并且，vertical spread 虽然存在卖出期权这个操作，但你并不需要任何保证金 (margin) 就可以实施这个策略，股价跌超或涨超行权价，你都不用承担额外损失，这点与上期提到的裸卖看跌/看涨期权 (sell naked put or call) 不同。现在市面上大部分的券商也能够做到将一个 vertical spread 策略算作一笔佣金 (commission)，虽然本质上你是买卖了两个期权。因此，vertical spread 这个策略也是美股财报“赌徒”们最常用的一个策略。

以上这三种期权策略是传统的赌财报策略中，美投君认为对于普通投资者来说可行性最强的三种。既然说是“传统”的赌财报策略，估计有些看官也猜到了，必然还有“非传统”的赌财报策略。传统策略中，投资者不管玩出多少花样，总有一件事是不变的，那就是你总得先猜个涨跌，也就是得先判断方向，才能具体做出操作。然而，一旦方向判断失误，就很可能得亏钱。还有另一大类赌财报策略，是完全不需要判断涨跌，或者不太需要判断涨跌，单单靠财报前后期权价格的波动来赚钱的策略。说白了，就是靠割传统赌财报方式的“韭菜”来赚钱。
