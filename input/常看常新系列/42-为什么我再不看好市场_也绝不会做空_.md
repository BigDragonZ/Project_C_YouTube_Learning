# 为什么我再不看好市场，也绝不会做空？

## 精修内容

最近，在我的会员网站上，总有人问我，既然我不看好短期的市场，为什么不做空 (Short Selling) 呢？在现在这种市场环境中，做空不是更赚钱吗？为何我还要一直保持做多 (Long) 呢？还经常有人因此说我过于乐观，有人说我死多头 (Permabull)。我倒不介意大家这么说，不过我也觉得我有必要跟大家分享一下我的投资思路。

我不做空跟我个人的投资思路有关。熟悉我的人应该了解，我是崇尚长期价值投资 (Long-term Value Investing) 的，而长期价值投资的一个前提就是做多。因为股市长期是上涨的，优质公司也是如此。如果我们能够抓住价值，即便短期会有这样和那样的波动，但长期赚钱是大概率的。而做空则需要很强的短期投机 (Speculation) 能力，并不是说投机不好，而是投机的难度较大，与价值投资的理念背离。

我不做空股票，最主要的一个原因是做空的容错空间太小了。即便我再确定短期的股价走势，我也可能会犯错。只要我有 1% 的不确定，我就不会去做空。因为一旦我做错了，股价没有跌，那我就一点回旋的余地都没有了。因为一旦股价没跌，我不可能期待股价长期一定会跌下去。但是做多不同，即便短期股价没有按照我想象的方向走，对于优质的股票，我也有信心它终有一天会回归。所以我能拿得住，即便犯错，我也有回转的余地。

有人说，如果我确定股价长期一定会下跌，比如我就是特别厉害，提前看到了危机来临，那能不能做空呢？我还是不会去做，因为我不确定自己有没有能力等下去。在持有空仓 (Short Position) 的过程中存在爆仓 (Margin Call / Liquidation) 的风险，尤其是当股价暴涨的时候。有的时候，即便你大方向判断对了，一次股价的暴涨可能就能让你没有资本继续做空下去。没有一定的资金量，我们很难耗得起。但是做多就不同了，短期的浮亏并不会使我整体的持仓增加太大的风险。在风险可控的环境下，我就有底气继续等待。

那么，要是股价马上就跌的话，是不是就能行了呢？还是很难。即便股价真的马上就跌了，我也赚了钱，那么什么时候撤掉做空、什么时候开始做多，都还是有很多犯错的可能，这也会消耗我们很多的时间和精力。我不可能像持有做多仓位 (Long Position) 一样，赚钱的股票就长期拿住即可。而且，短期的空仓过多，还很有可能会引发轧空 (Short Squeeze) 现象。像去年很火的 GME、Bed Bath & Beyond，就是在被轧空的情况下，出现了动辄就翻倍的暴涨情况。如果你这时持有做空仓位，那几乎不可能不爆仓。

所以，不是说不能做空，而是做空的难度太大，容错空间又太小。作为投资者，我们肯定不可能不犯错，重要的是避免不可挽回的损失。而做空就是那个能让我承担不可挽回的损失的交易策略。我自知没有那个能力，也不愿意花那些时间和精力。我所追求的是安心的投资，所以我我不做空。

最后多说一句，虽然我不喜欢做空，但不代表我认为市场就是要涨。不做空也不代表我不用看跌的策略。在当前的市场环境中，我会大量使用看跌的期权策略 (Bearish Options Strategy) 来给我的持仓做好对冲 (Hedging)。
