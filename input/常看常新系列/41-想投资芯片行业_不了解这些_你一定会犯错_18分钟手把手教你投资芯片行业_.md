## 精修内容

芯片行业可以说是美股上最赚钱的行业之一，在过去十年的股价远远跑赢了纳指和标普 500。但与此同时，它也是最难投资的行业之一。行业内牵扯着大大小小各种公司，你可能知道英伟达 (NVIDIA)、AMD、台积电 (TSMC) 这样的企业，但他们不过是行业内的冰山一角。行业内还有众多优质企业，他们相互配合，相互竞争，形成了大量不为人知的投资机会。即使是那些你耳熟能详的公司，他们的行业地位和投资逻辑，你也不一定都了解。

一个芯片从无到有，大致就可以分为图中的三个阶段：设计、制造和封装测试。芯片一开始都需要有人先做设计，这其中最具代表性的公司包括英伟达、AMD 和英特尔 (Intel)。设计完成之后，他们会把设计图纸交到工厂去制造。由于芯片制造的工艺都非常复杂，一般公司也都是找专业的公司做代工，比如台积电。不过也有少量的公司会自己把设计和制造都包揽了，比如三星 (Samsung) 和英特尔就是自己设计、自己制造的芯片公司。在制造之后，还有一些后续的工作，包括封装和测试，这步做完后就大功告成，可以供应给终端企业了。

上面说的是大的框架，在每个领域中还需要众多上游公司来提供支持。这里面的公司就相对比较冷门了，不过在这里面也不乏一等一的好公司。就比如给芯片设计公司提供软件支持的企业，Synopsys (新思科技) 和 Cadence (楷登电子)，他们都是护城河极深且增长迅猛的公司。在今年这种大熊市中，这些公司的股价竟然还创了新高。而给芯片制造和封装测试提供设备和材料的公司中也有不少大牛股，像大名鼎鼎的光刻机公司 ASML (阿斯麦) 就是其中之一。除此之外，还有 Lam Research (拉姆研究)、Applied Materials (应用材料) 等众多优质股票。这些公司在自己的细分领域里都是当之无愧的王者，也各自有着不同的投资逻辑。

接下来就挨个地给大家分析一下，芯片行业中不同领域的公司以及他们的投资逻辑。

咱们先来看芯片从无到有的第一步，芯片设计。就像造房子一样，你要想造个房子，那第一步肯定是要先设计好草图，把房屋格局、水电**路**什么的都先设计好。做芯片也是如此，第一步也是需要工程师把芯片的系统和逻辑先画明白，做成物理版图，然后才去制造。而芯片设计领域，做的就是这个事情。在设计领域中，最重要的两类芯片分别为 CPU 和 GPU。

CPU (Central Processing Unit)，又叫中央处理器，它就像是电子产品的大脑，甭管你有什么功能都离不开 CPU 的运算。虽然 CPU 的用途很广，计算能力也很强，但问题是，它只能算完一个再算另一个。而当需要同时运算的时候，CPU 这脑子就转不动了。这就需要 GPU (Graphics Processing Unit) 登场了。GPU 俗称显卡，如果说 CPU 擅长控制和复杂运算的话，那么 GPU 擅长的就是简单且重复的运算。别看它干的事听起来有点 low，但像人工智能、区块链、数据中心这些最前沿的技术，需要的都是这种同时进行大量运算的能力。由于 CPU 的应用场景已经较为成熟，而 GPU 的应用场景则集中在当前最前沿的科技上，所以在未来，GPU 增长的潜力会远大于 CPU。根据第三方网站的估计，GPU 的市场规模在未来五年内，每年能够增长高达 33%，这个增速已经非常恐怖了。

目前在 CPU 市场上只有两个核心玩家，分别是英特尔和 AMD。从市场份额上看，英特尔多年来一直占据着绝对的领先地位。不过受限于芯片制造上的工艺，近两年英特尔的蛋糕正在逐渐被 AMD 蚕食。另一大领域 GPU 则有英特尔、AMD 和英伟达这三家公司涉及。不过在高性能的高端独立显卡上，则只有英伟达和 AMD 这两家公司在做。从市场份额上看，英伟达始终保持在 80% 以上，有着绝对的优势。AMD 之所以还能分掉 20% 的蛋糕，是因为他们走的是性价比路线，在性能上，目前它还无法与英伟达进行竞争。

可以看到，在芯片设计领域，垄断性还是很强的，就这么几家公司在做。这就意味着作为其他需要芯片的企业，你必须要依赖这几家公司的芯片才能够生存。但是一些财大气粗的大科技公司可就坐不住了，包括亚马逊 (Amazon)、谷歌 (Google)、苹果 (Apple)、特斯拉 (Tesla) 在内的大公司们，现在都在努力地自己研发芯片。这里面做得最成功的是苹果，它自研的 M1 和 M2 芯片已经投入使用，并且效果很好，现在已经完全取代了之前的英特尔芯片。所以可见，未来该领域的竞争会更加激烈。

另外，除了 CPU 和 GPU 之外，芯片设计领域还有很多其他种类的芯片存在，其中也有不少优质的公司。比如做模拟芯片的德州仪器 (Texas Instruments)，专门做电动车和自动驾驶芯片的 NXP (恩智浦) 和 onsemi (安森美)，以及做内存芯片的美光 (Micron) 等等。他们在各自领域内也都是有独一无二的优势。不过，相对于 CPU 和 GPU 来说，这些细分领域的市场份额以及盈利能力都相对较弱一些。

看完芯片设计，咱们再来讲芯片制造。正如房子的模型画好后，就要找施工队来建造一样，芯片的图纸设计好后，也需要找专门制造芯片的代工厂来造出芯片。芯片制造的工艺是非常复杂的，它的技术含量极高，甚至不比芯片设计难度小。衡量这个领域技术先进程度的一个典型指标是制程，制程越低，说明芯片的精密度越高，难度就越大。目前最先进的制程是台积电的 5 纳米芯片，预计年底前他们就有望实现 3 纳米的制程。

在这个领域中，有三家最具代表性的公司，他们分别为台积电、三星和英特尔。这三家公司的商业模式有很大的不同。三星和英特尔是既自己做芯片设计，也自己做芯片制造，算是自产自销。而又由于紧缺的芯片制造资源，他们也接其他公司的芯片制造订单。而台积电则是纯粹的代工厂，他们只专注于芯片制造。跟他类似的还有 GlobalFoundries (格芯) 和中芯国际 (SMIC)。

在芯片制造领域，台积电有着最先进的技术和最高的良品率，因而它也占据着绝对的领先地位，在行业内它分掉了 54% 的营收。而且那些最赚钱也最稳定的大单，基本上也都在台积电的手上，比如苹果、英伟达、AMD。而三星则排名第二，占了 16% 的营收，剩下的 30% 则被几十家公司分摊。

芯片的代工行业有一个反直觉的特点。别人都是随着规模的扩大，越做成本越低，而芯片代工则反过来，他们随着时间的推移，成本会变得越来越高。由于芯片设计会变得越来越复杂，做代工的成本基本上每四年就会翻一番。这样的成本增长也让行业内的玩家变得越来越少，只有烧得起钱的主才能真正留下来，这就天然地形成了一个寡头垄断的格局。十年前，最高端的代工厂还有五家，分别为台积电、三星、英特尔、GlobalFoundries 和联华电子 (UMC)。但 2015 年，GlobalFoundries 和联华电子就放弃了 14 纳米以下的代工，只剩下了那三家。而这两年，英特尔似乎已经开始逐渐掉队了，只剩下了台积电和三星这两家，未来这个趋势恐怕很难被打破。

在芯片制造完成后，最后一步就是封装和测试了。这部分主要包括芯片的切割、焊线、塑封，最后还需要测试，才得以出货交给具体的厂商。由于这一部分的工作在整个芯片行业中相对比较低端，没有太高的技术含量，而且也没有什么能够在美国上市投资的公司，所以这里我也就不具体展开讲了。

现在我们再来回到这张图，可以看到芯片行业的主线，设计、制造和封装测试，我们已经讲完了。现在就该聊聊这几根支线上的公司了。这些公司各有各的特点，虽然相对冷门，但绝对不比主线的公司差。我们不妨先来看看芯片设计的上游企业，EDA 和核心 IP，这里面的宝藏公司可是有不少。

在介绍公司之前，咱们有必要先了解一下什么是 EDA。EDA 全称 Electronic Design Automation，即电子设计自动化。简单来说，就是给设计公司提供的软件支持。你芯片设计肯定不可能拿纸跟笔一笔一笔地画，芯片设计如此复杂，必须要有专业的软件配合才能够完成。而 EDA 就是专门提供这类的软件支持，它几乎是所有芯片设计公司的必备软件。而芯片设计又是芯片从 0 到 1 的关键，因此 EDA 也被称为是“芯片之母”。前不久，美国禁止给中国出口 EDA，就是因为盘算着中国没有了 EDA，几乎再无设计出芯片的可能。可见 EDA 对于整个芯片行业的重要性。

在 EDA 领域，最重要的公司是 Synopsys 和 Cadence。他们本质上是做软件的 SaaS (Software as a Service) 公司，因此也有了 SaaS 公司的特点，他们的毛利率特别的高。比如 Synopsys 的毛利率就常年保持在 70% 到 80% 之间。另外，他俩还有一个重要的特点，那就是受到芯片周期的影响较小。在经济不好，芯片卖不出去的时候，芯片设计公司们也不会彻底地暂停研发。而只要公司还在研发，还在设计，甭管你芯片卖得好不好，EDA 的软件该赚多少还是能赚多少。EDA 的这种稳定性也体现在了它的股价上。可以看到，Synopsys 和 Cadence 在今年大熊市的环境下，股价仅下跌了 4.9% 和 2.6%，而且在前不久还双双创下了新高。

另一个给设计公司提供辅助的重要业务是核心 IP。IP (Intellectual Property) 指的是芯片中具有独立功能的电路模块设计，这些芯片设计公司可以用它来减少设计的工作量。打一个简单的比方，就像是设计房子的过程中，你可以找一个现成的厨房设计方案，再找一个现成的卫生间设计方案，然后把它们拼接到你自己的设计中去，不需要所有的东西都从零开始设计。而 IP 企业就是给芯片设计公司们建的这种成熟的模块，设计公司只需要去买他们的成熟模块，然后在一定的逻辑下拼接起来就可以，这大大减轻了设计的工作量。核心 IP 中最重要的企业是 ARM，英伟达之前就想收购这家公司来着，但后来遭到了多国政府的集体反对，最终没有成行。可见核心 IP 公司在芯片领域的重要性。不过 ARM 这家公司并没有上市，我们也就不做过多的分析了。

最后，咱们再来看看辅助制造和封装测试的上游公司们，设备和原材料企业。这一块应该很好理解，建房子都需要有人来提供砖头和挖掘机，那芯片作为这么复杂、对精密度要求这么高的产品，肯定也少不了上游企业提供的材料和设备支持。在芯片制造的过程中有很多不一样的阶段，包括封装、沉积、测试、清洁，每一个阶段所需的设备和材料也都不同，而且都有很高的要求。诸如 Applied Materials、Lam Research、Tokyo Electron (东京电子) 的公司都在各自的芯片制造环节上提供着无可替代的设备和材料。不过，要说材料和设备这一块最大且最重要的一个公司，那毫无疑问当属光刻机制造公司 ASML 了。

光刻机是利用光在芯片上印制出微小图案的机器，这是芯片实现批量生产的一个基础，也是台积电和三星这样的芯片代工厂中最核心的设备。光刻机是全世界最复杂的机器之一，一台光刻机可以卖到 1.5 亿美元之多，但即便是这么贵，也是供不应求。因为最先进制程的芯片都必须要用到 ASML 的 EUV (Extreme Ultraviolet，极紫外光) 光刻机来做。ASML 的光刻机就关乎着这些芯片制造公司的生死存亡，谁有更多的 ASML 光刻机，谁就有更先进的芯片，谁就能掌握更多的话语权。因此，ASML 的机器根本就不愁卖，像台积电的购买计划已经考虑到 2024 年去了。

那为什么 ASML 能有如此强的议价能力和护城河呢？其实原本 ASML 也是有两个强劲的竞争对手的，这俩公司大家也都熟，就是日本的佳能 (Canon) 和尼康 (Nikon)。三十年前，当 ASML 宣布要孤注一掷研发 EUV 光刻机时，这两家公司都并不看好，所以也没有跟进。谁知经过了 20 年的努力，EUV 光刻机还真的就让 ASML 给做出来了，而且精密度是当时最先进的 DUV (Deep Ultraviolet，深紫外光) 光刻机的 1/15。因此，它马上就成为了最先进制程的第一选择，再也没有了其他竞争对手。自此，ASML 成为了世界上唯一一家有能力做 EUV 光刻机的公司。而强大的护城河也成为了投资这家公司最主要的投资逻辑之一。

到这里，芯片行业的业务逻辑就跟各位看官全部梳理完了。最后，咱们再来讲讲行业内，我认为最值得咱们投资者关注的两大风险。第一大风险是可能的摩尔定律的失效。根据摩尔定律，芯片的性能平均每 18 个月就会翻一番，这个定律在过去的几十年中一直成立。但不少专家都认为，随着人类即将在芯片上触及物理极限，摩尔定律也许很快就会失效。有人说是会在 2030 年，有人说是会在 2040 年。如果摩尔定律真的失效，芯片的性能将不再进步，那么对于整个行业发展来说将会是一大打击。

第二大风险是地缘政治的风险。芯片这个行业太过复杂，它必须要多个国家、多个企业非常精细化的合作才能够正常运转。而当前地缘政治风险正在不断增加。想象一下，如果台积电停摆了，芯片行业将会遭受怎样的重创。

到这里，芯片行业就给各位看官介绍完了。可以看到，芯片是一个玩家众多且极其复杂的行业，说是现代工业制造难度的天花板，我想也不为过。我想不少看官都知道，前几年，中国以举国之力大搞芯片设计和芯片制造，试图通过国家机器的力量打通芯片产业的难关。不过，最终这样的尝试目前被证明失败了，众多芯片项目搁浅，大量投研经费被白白浪费。

看完本期视频，我相信你也多少了解了其中的原因。别看芯片个头不大，看起来不起眼，但它比当年造原子弹的难度可是要大多了。各个环节相互制约、相互合作，才能够做起这么一颗小小的芯片。即便不谈协作，每一个单独的流程，难度也都是极高的。我们也不能仅仅将中国的芯片失败归咎于上游的“卡脖子”。ASML 花了 20 年磨一剑才搞出了 EUV 光刻机。同样的，中国要想在芯片领域上后来居上，在我看来，也需要带着这种钻研的心态才能成功。
