## 精修内容

最近的市场有点怪，不是没有坏消息，而是似乎多坏的消息，市场都跌不下去，还止不住地往上涨。不少人都表示看不懂现在的市场了。前段时间市场下跌的时候，大家都没这么慌张，这一下上涨了，投资者反而不知道该怎么办了。

在美股历次熊市 (Bear Market) 中，这样的反弹并不罕见，只是在牛市 (Bull Market) 真正到来之前，没人能够确定哪次反弹是彻底的反弹。

不过，有一点是可以确定的，那就是当前市场的不确定性依然很高。举个很简单的例子，不管你是看涨还是看跌，你现在敢孤注一掷地赌方向吗？我估计多数人还是不敢的，因为市场的风险并没有显著地降低。所以在现在的市场环境下，判断方向还不是最重要的，方向的对错也不是关键，最重要的是控制风险。

什么是应对风险最好的办法呢？在我看来就是期权 (Option)。很多人对于期权有误解，认为它是个高风险的投机品，实际不然。期权的诞生，最初的目的就是为了控制股票的风险，只不过是后来被很多投资者学废了，整出了太多投机的玩法。今天这期视频，就好好跟各位看官讲讲，如何利用期权策略来应对高风险的市场。今天要讲的这几个期权策略，不仅非常贴合现在的市场，还能让你在未来应对各种市场风险时都能够游刃有余。

用期权来控制风险的策略，也叫做期权的对冲 (Hedge/Hedging) 策略。在介绍具体的期权策略之前，我觉得有必要先跟各位看官，把对冲的正确思维给讲扎实。首先我们必须要清楚，我们做对冲是在对冲什么。很多人认为，对冲就是为了减少亏损，实际这是一个很大的误区。对冲最主要的目的，是为了对冲风险，或者说是降低你持仓的不确定性，减少亏损只是对冲的其中一种表象而已。

所以，我们之所以去做对冲，并不是认为明天的市场会跌，所以用对冲来减少损失，而是因为当前市场的风险是我们所不能承受的，所以我们才会开始做对冲。这两点是有很大不同的，我建议大家暂停一下，感受一下这两点的区别。对于我个人而言，我现在依然保持对冲的原因，并不是因为最近市场涨了很多，未来有可能会下跌，而是我认为目前市场上的不确定性依然很高。通胀、加息、经济衰退等各种风险，仍然可能对市场造成好或不好的影响。在这一刻，我对风险的判断没那么重要，最重要的是，我清楚一旦任何一种风险爆发，结果都是我不能承受的。所以我需要保持对冲来应对这一风险。

不过，投资者也要清楚一点，对冲都是有成本的。在风险环境下，用对冲成本来换取风险的降低，那是值得的，但如果风险已经降低到了可控范围内，这时再持续保持对冲，就只会限制你的收益了。作为长期投资者，最大化收益是你每时每刻都需要思考的事情，而不可控的风险只是在特定情况下才需要特别操心的问题。你可能会听到某些投资高手说，他们时刻保持对冲，你可千万别照猫画虎。他们时刻保持对冲，也许是因为他们有太高的杠杆 (Leverage)，时刻需要较强的风险控制；也许是因为他们做的期权交易本身就自带风险对冲；也有可能是他们压根就不懂对冲的真正含义。所以，我们美投君的观众，即便上了对冲策略，也需要时刻留意风险的走向。如果你自认是投资新手，还没有对于风险的评估能力，那么你最好也先不要上对冲策略，减仓或者是调仓也能够达到降低风险的效果。

现在你了解了对冲的思路，也认为自己有能力进行对冲操作的话，那么接下来就可以开始了解一些期权的对冲策略了。首先，我们来介绍一种非常适合新手上手的对冲策略——Protective Put，之后为了方便介绍，我们就简称为 PP。PP 是一种强对冲策略，非常适合于现在的高风险市场。在过去半年中，PP 也是我自己投资中主力的对冲策略之一。

PP 策略有两部分组成：一部分是持有一百股的正股 (Underlying Stock)，也就是你要对冲风险的标的；另一部分是买一个看跌期权 (Put Option)。我们知道，买 put 就是看跌，股票跌你就赚钱，而持有一百股正股，很明显就是看涨，股票涨你才能赚钱。这样，在股票下跌的时候，put 的收益就可以为你弥补一部分股票下跌的损失；而当股票上涨时，你买 put 的钱多半就作废了，这就是你的对冲成本。PP 策略就非常类似于你买的一种股票下跌保险，股票下跌时它能保你跌到一定程度就不会再继续亏钱了，但如果股票上涨，你这个买 put 的保费也算是作废了。

我给各位看官举个简单的例子，你们就明白了。假设我有一百股的苹果股票，股价是150美元，如果我担心苹果短期内会面临无法承受的风险，那么我就可以买入一个 put option 来对冲我这一百股的正股。假设我买入了一个一个月后到期 (Expiration Date)、行权价 (Strike Price) 145美元的 put，花费了我3美元的权利金 (Premium)，这就意味着这张 put 就可以帮我对冲股价在145美元以下的所有风险。什么意思呢？一个月以后，苹果股价如果跌超了145美元，那么跌超的那部分我就不会承担任何的损失，也就是说我最多就是亏到145美元，再多亏的部分就会由这张 put 的收益来自动弥补。这就像是保险公司的赔偿一样，一旦保险条件触发，保险公司就会全额负担你的损失。但是，如果股价上涨了，或者没有跌超145美元，那么我们买的这张 put 的3美元就全部打了水漂。

可以看到，这个策略的思路并不复杂，就是用一张 put option 来对冲我们整个的正股。你设定的行权价，就是你要保的、你能承担的最大下跌幅度；到期日，就是你要保护的时间。而且它还有一个特别大的优势，那就是它的对冲效果非常强，股价跌破行权价之后，任何的损失它都能够弥补。正是因为它思路简单、对冲效果又好，所以它是非常适合新手上手的对冲策略。

不过，PP 策略的防御能力虽然强，但它也有一个不可忽视的问题，那就是贵。这就跟买保险一样，你买保险就是为了图个心安，但如果保费太贵，你可能宁愿承受这个风险。PP 也是如此，买 put 的成本还是很高的，一旦股价没有下跌，或者跌幅不够，这笔期权费也会是一笔不小的损失。

那么，有没有什么办法，能够保持对冲强度的同时，还能够降低这个成本呢？还真有，这就是我们接下来要讲的策略——Collar 策略 (Collar Strategy)。Collar 相对来说要比 PP 策略稍微复杂一些，不过理解起来也并不困难。上面说了，PP 策略就是在持有一百股正股的同时，再买一个 put 来对冲。而 Collar，则是在 PP 策略的基础上，再卖出一个高位的看涨期权 (Call Option) 来为购买 put 的开销融资。所以，Collar 是由三部分组成的：一个是持有一百股的正股，一个是买 put 来对冲，最后一个是卖 call 来降低对冲的成本。而当你买 put 花的钱和卖 call 得来的期权费相同时，你就可以做到零成本对冲了，而且对冲的效果相比 PP 来说没有打任何折扣。

我们还是用一个具体的例子来说明一下 Collar 的用法。还是假设我现在持有一百股的苹果股票，当前股价150美元。同样地，我花费了3美元买了一张行权价145美元的 put option 来做对冲。与此同时，为了降低成本，我可以再卖一个行权价155美元的 call option 来获得3美元的期权费。这一买一卖，相当于是我没有花费一分钱，却有了一个145美元的 put 来帮助我做强对冲。如果风险真的来临，苹果股价暴跌了，那么我最多也就亏5块钱，股价在145美元以下再继续下跌，我就不会承担任何亏损了。

这时肯定有看官会问，我们凭什么能够多卖一个 call 呢？这对我们整体的对冲策略没有影响吗？当然有，卖 call 这个决定是需要有一定的判断的。我们之所以敢再卖一个 call，是因为我们认为这时的市场上涨空间是有限的，也就是说，我们认为股价短期内不会出现大幅度的上涨。如果股价最终没有涨超你卖 call 的行权价，那么 Collar 确实就是零成本对冲了你的标的。但如果股价涨超了卖 call 的行权价，那么涨超行权价的那部分收益你就拿不到了。所以理论上讲，Collar 并不是完全没有成本的对冲，你的成本就是牺牲的那部分上涨潜力。

用 Collar 策略来做对冲，你相当于是用牺牲的那部分上涨潜力去换取了一个强力的对冲保护。在持有 Collar 策略时，这种所谓的零成本对于你来说也是有很大帮助的。因为，除了股价暴涨这种特殊情况之外，其他情况，诸如股价小涨、不涨、小跌，对于你来说都没有任何影响，就跟你持有正股是一样的。而一旦股价真的大跌了，那么这个策略就能够发挥它最极致的作用，股价跌得越多，你能弥补的损失就越大。这很符合我们普通人的心理，股价小涨小跌，一般来说我们也都不太在乎，这时候能够零成本地有一个意外下跌的保险，我们持有的时候也能够更加踏实，这就是对冲的真正目的，让我们更为安心地投资。

在我看来，Collar 这个策略是很适合当前这种市场环境的。别看过去一段时间市场涨了不少，但是我认为，现在的市场还不具备开启下一轮牛市的土壤，短期内继续大幅上涨的空间正在被压缩。相反，市场的风险依然没有显著的下降，任何一个财报暴雷，或者任何一个数据报表都可能会引发市场新一轮的下跌。比如六月的通胀数据过后，市场就在一周内大跌了10%。当然，我并不是说这种事情一定还会发生，而是说这样的风险是我自己想要尽量杜绝的。所以，对于我个人来说，我是愿意现在牺牲一部分的上涨潜力来换取那个强力的对冲保护的。

在实际操作时，我们还可以通过调整 call 和 put 离当前股价的距离，来适应你对于股价的判断，以及你自己的风险承受能力。举个例子，比如我风险承受能力高，我可以承担每股10美元的亏损，再多我就不愿承受了，那么我就可以买140的 put，然后卖160的 call，这样我也能获得更高的上涨潜力。亦或者，我觉得苹果最近虽然上涨乏力，但是区间变动不会仅仅是5块钱，很有可能是15块钱，比起下跌保护，我更在意不能错过任何的涨幅，那么我也可以卖165的 call 来融资135的 put。再有，如果你不追求零成本对冲，也完全可以选择上下距离不同的行权价，比如165的 call 配上140的 put，这样你会花一些钱来做对冲，但也能起到用卖 call 来降低买 put 成本的效果。

这个 Collar 策略，你可以应用在个股上，就像上面苹果的例子那样，也可以用在你整体的持仓上，那样的话，你期权标的选的就应该是大盘，一般就是 SPY 或者 QQQ 这样的大盘指数。这时你选择对冲强度就更多的是从大盘的角度来考量了。比如，你能接受大盘最多跌多少？比如大盘从现在这个水平再跌超过10%，我就接受不了了，那么我就可以做一个大盘10%的 put 保护，再相应地选择卖 call 来降低这个成本。亦或者，你可能会认为大盘短期之内很难再重回高点了，那么你就可以先卖一个大盘高点的 call，然后再相应地选择 put 来选择你的保护力度。

现在为了降低 PP 这个策略的成本，进行了一步变形，到了 Collar 这个策略。其实 Collar 策略还可以再做变形，进一步优化期权对冲策略的效果。我们仔细看上面这两个策略，其实不管是 PP 策略还是 Collar 策略，都有一点点浪费，不知道各位看官注意到没有。像我们买 put 做对冲，本质上就是保护住了股价跌超行权价的全部亏损。就拿苹果那个例子举例，我们是保护住了苹果股价从145美元跌到0的全部潜在亏损。但实际情况下，苹果股价真的会降到0吗？恐怕很难，能跌个10%到15%就算是非常大的跌幅了。所以，我们其实可以不用保护全部的跌幅，而是只保护一部分，就能够满足我们的对冲需求了。要实现这样的效果，我们就可以在 Collar 这个策略的基础上，再卖一个更低行权价的 put 来进一步地降低成本。而这个策略也有一个名字，叫做 Seagull 策略 (Seagull Strategy)。

海鸥策略也是美投君自己最喜欢用的一种期权对冲策略，它在过去半年为我降低了不少风险的同时，也弥补了不少股价下跌的损失。海鸥策略实际上分成四个部分：一部分是持有100股的正股；一部分是买 put；最后这一部分是卖一个更低价位的 put。这个策略有点复杂，我们再举一个具体的实例。

我们还是用苹果这个例子。现在苹果股价150美元，我们持有100股的苹果正股。这时我们还是先买一个145美元的 put，花费了3美元的期权费。不同的是，为了降低成本，我们先卖一个140的 put 来获得1美元的期权费。这时我们的总成本已经从3美元变成了2美元。那么，因为这部分成本降低了，我们就可以卖出比原来行权价更高的 call 来弥补这2美元的成本。所以，相对于 Collar 中卖155美元的 call，现在我们可以卖出更高的160美元的 call 了。这样做，同样可以将整体的对冲成本降到0。这个操作相当于是变相地给我们留下了更高的上涨空间。

但 Seagull 相较于 Collar 的缺点在于，在股价下跌时，Seagull 策略不会保护145美元以下全部的跌幅，而是仅仅保护住145到140这个区间内的跌幅。股价从140美元再继续往下跌，我们还是会承担亏损。当然了，所有的这些价格都可以根据你自己的主观判断来进行调整，这里不过是给大家一个概念而已。

到这里，这三种用期权做强对冲的方法就介绍得差不多了。其实不管是 PP 策略、Collar 策略还是 Seagull 策略，最核心的对冲工具都是那张买入看跌期权 (long put)。我们接下来做的各种变形，别看花里胡哨，但实际上都是为了降低这张 long put 的成本，与此同时，尽可能地再留下更高的上涨空间。

不得不说，期权交易确实是有一定的门槛的，我这一期视频做不到事无巨细地教大家全部的期权知识。完全没有接触过期权的看官，我建议还是要谨慎地使用期权对冲策略。毕竟降低投资风险的方法不止期权这一种，我们也没必要为了降低风险而去接触一个自己都不了解的工具，反而增加了投资风险。
