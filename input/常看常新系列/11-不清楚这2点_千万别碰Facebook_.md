## 精修内容

最近一段时间，Facebook 一直处于风口浪尖之上。先是最开始的反垄断，然后是苹果 iOS (Apple iOS) 14.5 的更新对广告业务的打击，紧接着又有吹哨人谴责公司为了利益忽视青少年健康问题。同一时间，还出现了长达 6 小时的史上最大规模的宕机。大量的负面消息接踵而至，Facebook 的股价也从高点下跌了近 15%。它的估值也跌到了只剩 24 倍，便宜得跟麦当劳都差不多了。

对于 Facebook 这家公司，市场上也有很多争议。有人认为，它的企业文化有严重的问题，公司形象越来越糟糕，完全不值得投资。也有人认为，再多的负面消息也无法撼动它社交龙头的地位，也不会影响它的赚钱能力，所以每次下跌都是买入的机会。这些争论看似都有道理，但在我看来，都没有切中 Facebook 这家公司的要害。

我不会像之前分析公司那样，把 Facebook 的业务发展和财务状况掰开了揉碎。这些东西对于 Facebook 这家公司来说太表面，无法帮助我们做投资决策。我们要说两个对于投资 Facebook 这家公司来说有决定性的因素，并把这两点说深说透。中间也会有很多我个人非常主观的看法。

为了引出我们要讲的第一个决定性因素，有必要先梳理一下最近 Facebook 所遇到的几个问题。虽说 Facebook 最近丑闻不断，但是也是有主次之分的。像大面积的故障宕机就属于影响较轻的一类，人们很快就会忘掉这类事件的影响，也不会对股价造成太大的冲击。对于 iOS 的影响，有投行评估，更新的隐私条款只会影响 Facebook 所有广告投放中的 10%，而在受影响的这 10% 的广告投放中，最终的营收影响就更小了。而 Facebook 作为社交龙头，拥有极强的议价能力，Facebook 完全有能力将这部分影响转嫁到商家头上。因此我认为，长期影响也不会很大。

但吹哨人的爆料则是另一层维度的担忧了。并不是说吹哨人的爆料本身有多大的冲击力，这些事件最终也很有可能会草草收场。但这些事件所牵扯出的 Facebook 企业文化的问题，是足以左右投资决策的。这也就是我们第一个要谈的 Facebook 的决定性特质，那就是，它是一家为了理想而不择手段的公司。

回看 Facebook 的发展史，它的每一步发展似乎都在诠释着“不择手段”这个词。Facebook 的诞生就是扎克伯格 (Mark Zuckerberg) 靠公开哈佛 (Harvard) 女生的照片而一炮走红。之后 Facebook 社交媒体的雏形，也是踩在哈佛校友的原创设计之上而发扬光大的。这一路对于扎克伯格漠视隐私、涉嫌抄袭的控诉就从未停止过。而 Facebook 发展起来后，扎克伯格的这种风格似乎也从未收敛。类似侵犯用户隐私、影响选举、漠视种族歧视的丑闻层出不穷。最严重的一次是 18 年 Facebook 卷入的剑桥分析 (Cambridge Analytica) 的丑闻，Facebook 滥用用户个人数据，协助剑桥公司散布政治广告。此次丑闻一度让 Facebook 的股价下跌 40%，但 Facebook 的回应却始终不温不火，这更引起了舆论的愤怒。

可以看到，Facebook 不择手段是真的不择手段，但不可否认的是，扎克伯格也真的是一个非常有理想的人。如果你多观察他的言行会发现，他并非是为了利益而不择手段，而是为了理想。至少我是愿意这么相信的。扎克伯格的理想是非常宏大的，而且他会为此而付出行动。在个人修养上，他每年都会给自己定一个很难完成的挑战，比如说学中文、长跑、写 AI 的代码等。这说明他是一个非常注重自身认知进步的人。另一方面，在事业上，他想的也是如何要改变这个世界。十几年前，他想的是构建一个巨型的社交网络，改变人类的沟通方式，这个他做到了。现在他的理想是，构建一个人类通用的货币 Libra，或者进一步改变人类交互的方式——元宇宙 (Metaverse) 等等。

这样的扎克伯格给我的感觉是，比起运营好一家公司来说，他可能更在乎这家公司在做的事，能不能改变世界。现在社会上众多的指责，确实会伤害到 Facebook 的形象和声誉，扎克伯格不可能不明白这一点。但问题是，他不在乎 (care)。他不认为这会影响到他去追求理想。换个角度来理解，他可能认为，现在社会上的这些指责，在他的分析中，还没有到阻挡 Facebook 发展的地步。一旦事态到了影响 Facebook 实现理想的地步，我相信他也会全力以赴地去解决这些问题。

现在来看 Facebook 这一路的发展和扎克伯格为了理想而与公众为敌的处境，实际上像极了 20 年前的微软 (Microsoft) 和比尔·盖茨 (Bill Gates)。现在我们知道比尔·盖茨是一个对自己要求极高，并且非常有社会责任感的人，他的很多慈善项目都是旨在让这个社会变得更好。但在互联网诞生之初，比尔·盖茨也是因为抄袭和垄断等各种问题，一度成为公众的众矢之的。微软也一路野蛮生长，为达目的不择手段。当初大火的 Windows 操作系统，靠的就是抄袭苹果的窗口设计；曾经不可一世的网景 (Netscape) 浏览器，也被微软捆绑销售 Windows 和 IE (Internet Explorer) 浏览器的流氓操作给打得退出了历史舞台。然而，木秀于林，风必摧之。后来微软遭遇反垄断诉讼，此后便经历了长达十几年的蛰伏，股价也一蹶不振，直到近几年才有所恢复。

其实，如果用今天的眼光来回看这段历史，虽然依然能够看到微软自身存在的问题，但不可否认的是，微软的遭遇也和当时社会对于互联网的认知不够有很大关系。微软当时最为人所诟病的垄断问题，除了人为的因素，其实也有一部分原因是和互联网技术天然的垄断性有关。这是身处新兴领域的互联网企业在发展时所不可避免的阶段。可惜，当时的社会对于互联网的认知还不够，或者单纯地因为对于强大且未知事物的恐惧，导致微软在极盛时被压制，不得不被那个时代人的认知不足所束缚。

反观当今的 Facebook 又何尝不是如此。别看我们现在天天都在用社交软件，但它真正出现在人类社会中，也就十几年的时间。社交网络牵扯的是一个又一个的人，有人的地方就会有人性的丑恶，就必然会带来大量关联的社会性问题，这是任何一个社交平台都无法完全避免的。包括 Twitter、Snapchat，中国的微信 (WeChat)、微博 (Weibo) 都或多或少因为这些问题被舆论口诛笔伐。咱们人类现在还在学习怎么去适应社交媒体所展现出来的人性丑恶，怎么去处理社交媒体给社会带来的种种问题。而人们对于 Facebook 的指责，或者说是恐惧，很大一部分原因就是因为现在人们还没有搞明白怎么去处理社交平台和社会道德之间的联系。这和 20 年前我们因为搞不清互联网对于人类生活的影响，而将矛头对准微软是一个道理。

当然了，我说的这些也不是为了给 Facebook 洗白，它确确实实在企业文化上有很多的问题，这个锅还得 Facebook 自己来背。我想表达的是，为了理想而不择手段的 Facebook，也许真的有什么远大的理想，也许真的想出了在实现理想的这条道路上可以舍弃什么。但是，现实往往不是理性的数学推导。在人们没有彻底搞清楚如何和社交媒体相处之前，对于强大且未知的恐惧，就很可能会让 Facebook 经受它自己以及自认理性的投资者们所始料未及的风险。也许 20 年后，我们再回看今天的争论会觉得幼稚，但如今，这就是一个实实在在的风险压在 Facebook 的头顶。

关于 Facebook 企业文化的问题，我们就先聊到这里。抛开这些问题，回归到公司当前的业务和发展上面，我们到底又该如何理解 Facebook 这家公司呢？这就要说到第二个我认为投资 Facebook 的决定性因素了。那就是，它是一家具有恐怖潜力，但却仍未开发的一家萌芽公司。注意，这里的关键不在于潜力，潜力每家公司都有，分析潜力不值一提，重点在于仍未开发。这是一种无声胜有声的威慑力，也是一种无限的想象。

我说这话可能有人会质疑我，Facebook 目前已经万亿市值，拥有全球人口一半以上的用户，它还有什么潜力，又怎么能说未被开发呢？别急，我带大家一点一点拆解 Facebook 这家公司的深层逻辑。

要想理解为什么说 Facebook 的潜力还未被开发，我们需要一头扎进它的财报，去发现那些容易被人忽略的蛛丝马迹。Facebook 的财报和其他很多大科技公司有很大不同，很多地方甚至可以说是令人费解。咱们先来看它的利润表 (income statement)。可以看到，这家公司有 98.5% 的营收来自于广告。虽然说在强大的用户数据支撑下，它的广告营收增长非常地快，能够达到 50% 以上的年化增长，但是，对于这样一家全球前十大的科技巨头来说，只靠广告挣钱，这个盈利模式未免也太单一了。

再来看它的资产负债表 (balance sheet)。这里有一行数据非常令人费解，就是长期借贷 (long-term debt)，Facebook 这家公司几乎是从不借钱。要知道，这对于一家公司来说并不是什么好事，尤其是在当前极低利率的环境下。不借钱，说明公司的杠杆率过低，有很大的发展潜力没有被释放。如果你看苹果、微软、亚马逊 (Amazon) 的财报，哪个不是利用自己的影响力，以极低的成本借个百亿千亿的钱来发展项目。而 Facebook 明明也有能力这么做，但却丝毫没有。

那不借钱的话，Facebook 的现金流会不会吃紧呢？要知道隔壁 Netflix、特斯拉都是在借钱借到天花板的情况下，仍然现金流吃紧的公司。这样，咱们再来看一下它的现金流量表 (cash flow statement)。可以看到，反映公司经营状况的营业现金流逐年呈指数性增长。别说吃紧，富裕到花都花不完。Facebook 每年需要几百亿几百亿地买国债来管理现金。

那么问题来了，Facebook 这么多年到底在干嘛呢？实际上，这么多年它基本上就只干了一件事，那就是积攒社交网络的用户。Facebook 的月活用户数一直保持稳定增长，十年间翻了三倍还多。别小看拉人头这事，这背后是深耕整个社交领域的布局以及庞大且高质量的用户数据。照理说 Facebook 有如此大的影响力，但只靠广告赚钱，听起来似乎有点不合理，但实际上，Facebook 的整个运营策略跟互联网公司的发展逻辑有很大关系。

这就要说到互联网行业中著名的漏斗理论了。在互联网公司的发展中，有且仅有两种发展模式。第一种是尽可能地获取用户，等规模做大了，再进入下一阶段，利用现有用户和数据进行变现。比较典型的案例是谷歌 (Google) 和微信。另一种发展模式是在一开始就寻找各种变现方式，然后通过营销手段形成良性循环，打开更多的用户群，从而发展壮大。最典型的例子就是亚马逊、优步 (Uber) 和阿里巴巴 (Alibaba) 这样的公司。就像这张图所展示的，第一种模式，先吸纳用户再变现的方式，就是从漏斗的大口往下发展到小口的模式。第二种模式，先找变现方式再扩大用户群，就是从漏斗的小口往上发展到大口的模式。

而 Facebook 很显然用的是第一种模式。最关键的是，它现在仍在最上方的发现 (Discovery) 的阶段，也就是积攒用户、获取数据的阶段。它实际上还没有开始正经变现自己的用户。Facebook 的数据可以说是所有个人用户数据中，数据量最庞大、质量最高的。你喜欢什么内容，朋友是什么人，喜欢跟别人聊什么，甚至使用手机的频率和位置都被它背后强大的数据分析团队搞得一清二楚。它要是正儿八经地想变现这部分潜力，不管是是从数据质量，还是从公司潜在的收钱能力来说，都是市面上任何一家公司所无法比拟的。这才是 Facebook 这家公司最可怕的地方。

实际上，我们已经能够从 Facebook 的一些小试牛刀中，看出这种恐怖的潜力了。比如 Facebook 正在做的电商 Facebook Shops，就是一个很好的例子。Facebook Shops 能够根据你的个人喜好，制作最适合你逛的商店。令人难以置信的是，它仅用了一年半的时间，就积累了 3 亿的月活用户。要知道，亚马逊 20 年的积累，也不过是这个水平。这就是 Facebook 多年深耕社交领域、厚积薄发的结果。同样的，如果 Facebook 接着说，我想做点别的，好比说金融科技 (FinTech)，或者说元宇宙，甚至是一个还未被公开的项目，我相信，凭借它积攒的数据、人才、资源，以及公司仍未开发的融资能力，Facebook 也会是那个最可能获得成功的那家公司。届时，说它的价值能有现在的十倍，我认为也不为过。

说到这，相信有些看官一定会有这样的好奇：为什么 Facebook 手握如此恐怖的发展潜力，却一直搂着不释放呢？对于这个问题，我也没有一个明确的答案，我只能说说基于我认知范围内的一些猜测。从第一点 Facebook 和扎克伯格为了理想而不择手段的风格中，我们不难看出，变现对于扎克伯格来说，也许并不是那么要紧的事。是的，Facebook 现在就有很强的变现能力，它也完全可以撬动这些未被利用的资源来大规模变现。但是，这对于小扎实现所谓的改变世界的理想会有帮助吗？恐怕不能，也许还会消耗他实现理想的能量。所以我们会看到，今天 Facebook 在变现上保持了极大的克制，也许就是为了实现一个更大的理想。

但也不排除这是 Facebook 的一种韬光养晦的手段。当年微软为了避免再被说成是垄断，而选择了韬光养晦，刻意降低利润，减少收购。如今，Facebook 会不会是照猫画虎，为了避免舆论攻击它利用用户数据变现，而刻意选择收敛锋芒呢？这个就留给各位看官自己思考了。

相信说到这，是否投资 Facebook 这家公司，各位看官应该也有一个自己的判断了。该投还是不该投，没有一个正确的答案，但我相信，对于 Facebook 为了理想而不择手段的企业文化，以及手握恐怖潜力却仍未开发的经营风格，每位看官都会有自己的判断。我个人是愿意相信 Facebook 不至于被企业文化所束缚，最终能够实现它的理想的。
