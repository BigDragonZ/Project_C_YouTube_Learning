## 精修内容

AI 时代，什么是最具确定性的投资机会？有人说是 **英伟达 (Nvidia)**。但还有这么一个产业，它是所有 AI 大模型必备的基础服务商，是几乎所有科技公司要想应用 AI，必备的桥梁。如果说英伟达是 AI 的“挖金铲”，那么这个行业就是 AI 的“炼金炉”。这就是我们今天要介绍的主角：**云计算 (Cloud Computing)**。

## 云计算的黄金十年与 AI 带来的变局

过去十年，云计算可以说是全球科技领域最为成功的产业之一。作为互联网时代所有企业数字化的必备基础设施，云计算行业不仅需求巨大，而且长期维持着极高的增速。这也造就了当前市场上最强的几家科技巨头：拥有 **AWS (Amazon Web Services)** 的**亚马逊 (Amazon)**，拥有 **Azure** 的**微软 (Microsoft)**，以及拥有**谷歌云 (Google Cloud)** 的**谷歌 (Google)**。这种三足鼎立的格局，持续了近十年之久。

随着 AI 时代的降临，这一切很可能即将发生彻底的改变。这个变化就在最近才刚刚展露苗头，一旦发生，很可能将彻底重塑整个科技产业的格局。

### 什么是云计算？

简单说，它就是把算力、数据、存储这些资源放到互联网上，企业或个人不再需要自建本地机房，而是通过互联网去调取这些资源。相比于传统的本地计算，云计算就像是租了一台超级电脑，可以随取随用，省钱又方便。过去十年，“企业上云”是全行业的一个大趋势。根据 **Statista** 的数据，2017 年全球云计算市场的规模只有 500 亿美元，到了 2024 年就已经高达 3300 亿美元，年化增长 31%。这也造就了亚马逊、微软、谷歌这样的超级巨头。这个体量和增速，是过去十年最成功的科技产业之一。

## AI 重塑云计算的投资逻辑

随着 AI 时代的来临，云计算整个产业的投资逻辑正在发生改变。AI 大模型的发展需要大量的芯片算力，这是英伟达的业务；而这些算力则由云计算公司来整合和交付。**OpenAI** 不会自己购买芯片搭建数据中心，这是一个难以想象的大工程。他们必须借助云服务商，才能使用算力训练模型，并将模型推广到市场。

在可预见的未来，几乎所有的 AI 商业模式都离不开云计算的参与。例如，OpenAI 训练大模型需要大量的云计算服务；用户使用 **ChatGPT**，其推理过程也需要经过云服务商。未来 AI 应用大面积落地，无论是 AI 应用公司还是 C 端用户，使用 AI 都需要云计算。即便最终 AI 边缘计算时代来临，增加的本地计算也需要更多的云计算来进行整合。因此，如果相信 AI 注定会迎来爆发，那么云计算就是最具有确定性的投资机会之一。

### 数据验证：行业进入新一轮加速增长

从 Statista 的云计算行业规模数据中可以看出，自 2023 年 ChatGPT 大火以来，云计算行业整体开启了一轮新的加速增长。2025 年预计收入规模能够达到 4100 亿美元，增速 24%。三家龙头企业——亚马逊的 AWS、微软的 Azure 以及谷歌云——增速更快，整体能够达到 27%。在如此大体量的基础上还能够实现加速增长，已实属不易。

作为投资者，分析行业增长，最关键的不是看收入。收入是滞后指标，已经反映了现实情况，更应该看的是企业的投入，这才是前瞻指标。如果企业管理层敢投入资金，往往意味着行业未来有更大的增长潜力。几大云计算巨头，包括自建数据中心为开源 AI 模型提供算力的 **Meta**，在投入上都在加速增长。这意味着行业未来大概率还有不俗的增长空间等待释放。

所以在 AI 时代，云计算公司将会是一个极具确定性，又有不俗增长潜力的领域。这在当前美股 AI 赛道上，绝对是极其稀缺的机会。而对于云计算本身而言，过去这十年“企业上云”的趋势还在，现在又叠加了 AI 的全新动能，云计算企业的投资价值显然又被抬升了一个维度。

## 格局的颠覆：数据背后的信号

云计算行业并非新事物，就那么几家巨头，投资谁不一样呢？以前这么投资绝对没问题。但就在最近，行业发生了一些格局性的变化，恐怕将彻底颠覆整个行业的投资逻辑。

在全球云计算市场中，份额占比最大的是亚马逊的 AWS、微软的 Azure 以及谷歌云，他们分别占了 30%、21% 和 12% 的市场份额。这个格局在过去十年略有浮动，但三足鼎立的格局非常稳固。从收入和收入增速来看，行业老大 AWS 的增速在最近两年维持在 18% 左右，并且非常稳定；老二 Azure 的增速则高达 35% 以上，并且还在加速，最近一个季度达到了 39%，而且未来依旧能够保持这样高的增速；老三谷歌云，体量虽小但增速也不慢，也能到 30% 左右，未来也基本是这个水平。

再來看一下这三家公司未完成订单的情况。如果说收入是已经实现的增长，那么未完成订单就是他们未来的增长潜力。老大 AWS 的增速在持续地放缓，虽然最近两个季度有所恢复，同比增长 25%，但环比只增加了 60 亿美元，处在同期的历史低点。老二 Azure 则表现最为亮眼，2024 年之后触底反弹，同比增长了 37%，环比大涨了 530 亿美元，远超所有同行。老三谷歌云的表现也不错，目前增速 37%，创下了三年来的最高水平。

整体来看所有这些数据，整个行业在 2024 年之后出现了加速增長，这也符合前面分析的行业趋势。但更重要的一点是，这三家龙头公司正在发生一些微妙的變化。今天的重点就是来探究这变化背后的原因，从中能找到云计算行业未来全新的投资逻辑，以及这背后隐藏的真正机会。

## 深度对比：谁是 AI 时代的新王？

要想探究行业变化的深层原因，最好的一个办法就是对这几家科技巨头进行横向对比。由于谷歌云披露的数据实在太少，我们先拿 AWS 和 Azure 做个对比，最后再统一拿谷歌进行比较。

从刚刚的几组数据中，我们能够明显看出，AWS 虽贵为龙头，但增速开始出现放缓。而 Azure 不仅增速更强，而且从增量上看也超过了 AWS。这就不是基数的问题了，而是这两家公司在发展上发生了实质性的分化。

### 短期因素：产能竞赛

一个最直接的原因是产能。过去两年，生成式 AI 的爆发打了所有云计算公司一个措手不及，因为算力需求的增长实在太快了。而新建数据中心扩展算力又不是一件容易的事，所以一直以来，这些云服务商根本就满足不了这么大的市场需求，都面临着产能的问题。

不过，微软在数据中心的投入上可以说是这几家中最早也是最激进的。微软早在 2023 年便开始大幅增加它的资本性开支，这是所有大科技中动作最快的，并且在之后的每年，它的增量也都是所有大科技中最大的。这就给了微软在产能释放上的优势。我们也可以从折旧数据中看出一些端倪。因为光买 GPU 并不意味着这些 GPU 的算力就可以使用了，但一旦你真的开始使用，从财务上便会有折旧成本的体现。最近这两个季度，微软的折旧成本出现了一个飞跃式的提升，这或许意味着公司最近释放出了大量的计算产能，也许这就解释了为什么公司近两个季度的业绩增长。

反观 AWS，它的节奏就要慢得多，不仅开始得慢，而且甚至最近的算力投入还下降了。公司最新一个季度的投入出现了放缓，只有 160 亿美元。而且要知道，和微软主攻 Azure 不同，亚马逊还有电商业务要照顾，这 160 亿里面可能还得有不少要投入到电商的基础设施中，真正放到数据中心的投入恐怕还不及这个数。所以，其产能问题明显要更加严峻。在产能集体卡脖子的阶段，谁能率先释放出更多的产能，谁就更可能带来更好的增长，而微软显然走在了前面。

### 长期因素：大模型带来的“飞轮效应”

产能只是短期的限制，还有一个长期的变化将一直影响着行业的格局。如果只看传统云服务，实际上 AWS 和核心 Azure 是趋近于相同的，不仅趋势一样，增速也类似。AWS 在传统云计算领域的优势还是很大的，它真的好用，有着最专业、最全面的云配套基础设施，其领先地位其实很难被动摇。

过去十年三足鼎立的格局之所以稳固，就是因为此。但现在，一个史诗级的变化出现了：AI。AI 对云计算行业的促进是全方位的，而这背后的根本，就是大语言模型的应用。

在大语言模型上，微软的背后是大名鼎鼎的 OpenAI。微软跟 OpenAI 的合作合同要持续到 2030 年。这个过程中，OpenAI 的模型训练离不开 Azure，别人使用 OpenAI 也需要用到 Azure 的算力。而最为关键的，其实是微软的 Azure 有着 OpenAI 独家的 **API (Application Programming Interface)** 权利。也就是说，企业要想在云服务上调用 ChatGPT 模型，只能在 Azure 上使用。这就意味着 Azure 在 AI 应用层面形成了入口垄断。

这样，微软就可以利用这个独家的 API 使用权，将想要使用 OpenAI 技术的企业客户牢牢地锁定在自己的生态系统中。这些客户不仅仅会使用 AI 模型，还会同时购买 Azure 的其他云服务，这就形成了一套强大的飞轮效应：OpenAI 的吸引力越大，Azure 的客户群就越大；反过来，OpenAI 的进步空间就越高，那么微软的云市场地位就越牢固。

反观 AWS，他们背后也有一家大模型，叫做 **Anthropic**。Anthropic 的 **Claude** 模型被认为是市场上仅次于 ChatGPT 和 **Gemini** 的第三大模型，但它相对来说更偏专业，主打编程领域，潜在的用户群体会小不少。而最为关键的是，AWS 和 Anthropic 的合作并不是独家绑定，Claude 在 AWS 和谷歌云上都有 API 接口。即便是在编程这个专精领域，AWS 也面临着被分流的风险，对于用户也起不到任何绝对的杠杆作用。

从数据上看，Azure AI 在 2025 年最新一季度已经达到了 40 亿美元，并且保持着超过 800% 的增速。更重要的是，就在本季度，我们还看到了 AI 给 Azure 云带来的外溢效应。最新的季度数据显示，核心 Azure 云也迎来了加速的增长。管理层解释说，这是因为 **SAP** 的用户上云所带来的增长。要知道，SAP 的客户都是像雀巢这样的超大型企业，他们本次上云选择了 Azure，很大一部分原因也和 Azure 的 AI 服务有关。而 AWS，公司并未披露 AI 收入的规模，但分析师普遍预计不会超过 5%。

所以，Azure 和 AWS 在云计算收入增速上的差距，表面上看似乎是因为产能的限制，实际本质原因在于 OpenAI 和 Anthropic 这两大 AI 大模型公司的能力差距。微软依靠独家的 OpenAI 合作，不仅从 OpenAI 那里获得了大量的收入，还靠 OpenAI 绑定了更多的企业用户。可见，AI 给 Azure 带来的提升是全方位的。而 AWS 虽然也与 Anthropic 合作，但规模远不及 OpenAI，而且合作还并非独家。所以现阶段 AWS 还是以传统云计算为主的一种业务模式。这或许就是为什么 AWS 在资本投入上不够积极的原因，因为它暂时还没有看到足够确定的 AI 需求。所以说，在 AI 时代，谁掌握了最强的 AI 大模型，谁就掌握了云计算增长的核心动力，而微软显然是领先的那一个。

### 潜在的颠覆者：谷歌云

既然说到了大模型的重要性，那我们肯定不能绕过拥有 Gemini 的谷歌。谷歌 Gemini 的能力在近两年提升飞快，在很多领域它都可以和 ChatGPT 平起平坐了。如果我们上述的逻辑成立，那么谷歌云是否也值得期待呢？

由于谷歌这家公司的重心还是放在了它的搜索和广告上，在谷歌云上披露的数据远远不及上面这两家公司多，所以我们只能从一些定性的角度来进行分析。从客户的结构上看，谷歌云的主要用户主要分为三大类：
- **第一类**是像苹果、**字节跳动 (ByteDance)** 这样的超大型客户。
- **第二类**则是传统行业的大型客户，例如沃达丰、**美国职业棒球大联盟 (MLB)** 等。
- **第三类**则最为关键，他们是新兴的 AI 企业。根据谷歌自己的披露，近 90% 的生成式 AI 独角兽公司以及超过 60% 的获得投资的 AI 初创公司，都是谷歌云的客户，而这其中就包括 Anthropic 和 **Perplexity AI**。

谷歌能够吸引 AI 初创企业，其逻辑和微软完全不同。微软就像是曹操，“挟天子以令诸侯”，挟着 OpenAI 这个天子以令诸侯；而谷歌则更像是刘备，他靠的是软实力来招贤纳士。因为谷歌早早就布局了 **TPU (Tensor Processing Unit)** 这种专门为 AI 设计的芯片算力，也布局了 **TensorFlow** 这种 AI 的开源平台。在大语言模型还没有爆发的时候，谷歌就已经是 AI 家族中的领军者了。所以，现在 AI 初创企业已有很多在谷歌生态中待了多年，自然也对谷歌多一份信任。

现在市面上的 AI 应用公司还不成气候，所以也就形成不了真正改变格局的算力需求。但要知道，未来一定是属于 AI 应用层公司的，所以这股势力未来很有可能成为一股奇兵，冲进市场，颠覆现有的云计算格局。

谷歌在云上最近下了不少功夫，也已经看到了积极的改变。比如今年 5 月，谷歌云与 OpenAI 达成了合作协议。OpenAI 以前是独家使用 Azure 来提供算力，现在也开始使用谷歌云的算力。而 9 月底，谷歌云又和 Meta 签署了一份为期六年，价值超过 100 亿美元的算力合作协议。此前 Meta 主要依赖 AWS 和 Azure，现在首次将部分核心算力转移到了谷歌云上。

这里有个很有意思的现象，不知道大家发现没有？OpenAI 选择和谷歌云合作，这一点多少有些匪夷所思。站在 OpenAI 的角度，它宁愿选择有直接竞争关系的谷歌合作，也不和毫无利益冲突的 AWS 合作，那这是否意味着 AWS 在 AI 云服务的技术能力上实在是不行呢？

整体来看谷歌云的情况，我認為它雖然身位靠後，但是爆發力最強。因为它不仅有着 Gemini 这个不输 ChatGPT 的杠杆，同时还有着 AI 应用最好的生态。只不过，从现实的角度来看，这种潜力还没有到真正转化成业绩的时刻，投资者还需要耐心等待。

## 最终结论与投资建议

整体来看云计算行业的发展，我们不难发现，传统的“企业上云”的趋势，已经过了增长最陡峭的阶段。如果没有横空出世的 AI，这个行业很可能将逐渐走向成熟，失去高增长所带来的估值溢价。而 AI 的到来，则彻底改变了这个行业的投资逻辑。

在我看来，AI 时代所带来的云计算需求，是比互联网时代要更高、更持久的需求。而这种需求并不像 AI 应用那样镜花水月，它是现在就能实际转化为业绩的一种需求。不管是现阶段大模型的发展、未来 AI 应用的爆发，抑或是最终边缘 AI 的兴起，云计算都扮演着不可或缺的角色。因此，它也是我认为现今美股市场上极为罕见的兼具确定性和增长的 AI 投资机会。

但与此同时，AI 也彻底打破了以往的行业格局。现阶段，决定云计算行业增长的关键，不再仅仅是技术和规模，而是各家公司背靠的大语言模型。大模型不仅自身能够给云计算带来巨大的算力需求，同时也能作为一种杠杆，撬动云服务商的客户增长和客户粘性。所以，它已然成为左右现今云计算格局的主要胜负手。现在，微软背后是 OpenAI，谷歌背后是 Gemini，而 AWS 背后则是一个松散绑定的 Anthropic，各家的地位和杠杆一目了然。

### 投资首选：微软

现在三巨头中处境最好的显然是微软。它背靠现今最强大的大模型 ChatGPT，独家合约一直到 2030 年才结束。这个过程中，微软的增长就有了较强的保障。而且，公司发力也最早，早早地解决了产能的瓶颈，开始变现自己的杠杆。在我看来，如果微软保持这个趋势，Azure 超越 AWS 成为行业老大，很可能就是一年之内发生的事情。

但作为投资者，你还得清楚，微软一直以来的风格是“扮猪吃老虎”。和其他公司有一点优势就要大张旗鼓地宣传出来不同，微软往往是刻意隐藏自己的锋芒。像本次财报后，Azure 业绩大涨，微软管理层就特地强调，这样的增长和 AI 一点关系都没有。但我们刚刚分析了，这个肯定不是事实。但微软为什么还要这么说呢？这和公司的历史有关。2000 年，微软经历了一次伤筋动骨的反垄断调查，股价十年后才得以恢复。自那之后，微软就学乖了，主动去降低自己的利润，避免锋芒毕露。而现在也是如此，如果微软高调宣传 AI，那难免再被监管盯上。毕竟跟 OpenAI 签的独家 API 协议，是真的存在垄断风险。

反过来想，正是这种“扮猪吃老虎”的风格，才是我们投资者的最好机会。因为比起那些动不动就因为管理层画的“大饼”而被高估的公司来说，微软这种风格就容易让它的股价被低估，这也给我们长期投资者带来了更好的安全边际。

### 其他选择与风险

而另外两家公司，目前来看则各有各的问题。AWS 是在 AI 上的布局太晚也太不确定，现在还面临着产能卡脖子的问题。如果下半年产能有所好转，那么业绩可能会得以释放，但 AI 布局的问题得不到解决，长期的投资价值就一直是一个担忧。而谷歌则存在管理层重心不明确的问题。现在管理层的重点都放到了 Gemini 上。这确实也应该，毕竟 Gemini 的问题解决了，其他的问题，不管是搜索还是谷歌云，就都迎刃而解了。但 Gemini 要想真正反哺谷歌云还需要很长一段时间，云计算的增长不仅仅是技术那么简单，还需要大量的运营，而谷歌现在的资源不在这里。所以，谷歌的投资逻辑在 Gemini 上，而非在谷歌云上。

所以，如果让我选，我还是会选择微软进行布局。

最后，我们必须还要清楚一点，AI 改变云计算的，不仅仅是行业格局，还有行业整体的风险。正因为现阶段大模型的能力成为了最为关键的胜负手，反过来讲，它也说明行业的格局过度依赖大模型的能力了。以往三足鼎立的格局之所以稳固，在于每家公司的成功都依赖于自己的技术特点以及规模效应。但现在，竞争力反而要依赖于外部来实现了，每家公司的风险也就都变得更高了。现在确实是 OpenAI 领先，但未来谁能够最终领先呢？没人知道，甚至连这些云服务商自己也都不知道，这个风险是我们所有投资者都必须要承受的。
