## 精修内容

最近有一个问题一直困扰着我：在当前的市场环境下，我们还能投资什么呢？众所周知，现在的市场环境不好，通胀、经济衰退等各种担忧甚嚣尘上。

有人可能会说，去投资那些稳定的防御股比较好，比如像医疗、公共事业这样的股票。它们确实能少亏，但问题是也不怎么赚钱。如果投资那些暴跌的成长股，它们确实便宜，也可能出现了机会，但在当前这种市场环境下，谁也说不好它们还会跌多少。那投资能源和材料股呢？毕竟在高通胀的环境中，别的股票都跌，它们还能涨。这也确实可以，但问题是，这些行业不确定性非常强，一旦跌起来，可能就再也不回头了。

那么该怎么办呢？一个最能让人安心的选择，是去投资那些既能攻、又能守，还具有确定性的企业。下跌的时候，你不用担心它们会崩盘；反弹的时候，你也知道它们的收益潜力不比任何一家公司差。最重要的是，你永远都知道，长期来看你一定是可以信任这家公司的。只有这样的投资拿在手里，才能让我们安稳地度过眼下这个动荡的环境。

满足这样条件的美股公司凤毛麟角。不过，天下无难事，只怕有心人。我们可以详细分析这么一类股票：它既有如公共事业一般的超强防御性，又有成长股一样的高速增长潜力。衰退的时候它可以少受影响，复苏时还有极强的爆发力，而且它还属于我们投资者最爱的科技行业。这个行业，就是云计算（Cloud Computing）。

为什么说云计算是极少数攻守兼备的行业呢？这背后的原因可能会超出你的想象。科技公司听起来本身就和防御不沾边，云计算又会有什么不同呢？云计算行业的防御性非常强，比起电信公司或者医疗公司来说一点都不差，只不过很多人没有意识到而已。

想想我们传统意义上理解的防御性股票，它们的防御性体现在哪？本质上，它们的防御性来自于人们的“不得不买”。即便经济衰退，人们也得交电话费，也得看病吃药。所以，电信和医疗行业的防御性就非常强。

那么云计算公司呢？云计算是一种通过网络远程提供硬件资源的服务，这个服务包括算力、存储、数据库等等。说白了，云计算就是给公司提供了网络上必须的基础设施。公司不再需要自己建立数据中心、自己维护服务器，而是向云服务商交服务费，来通过网络获取这些资源。在当今这个互联网时代，云基础设施就像水电一样，是企业们不得不使用的服务。即便经济再不好，企业们砍掉什么预算也不能砍掉云服务的预算，甚至他们还会更多地使用云服务来降低自己的成本。这一点，我们已经从最近的财报季中看出了趋势：在众多公司都纷纷下调预期的情况下，为企业提供云服务的公司普遍表现非常坚挺。

那么，为什么云计算能有如此强的确定性呢？这就要说到它那些无法让企业拒绝的优势了。

云计算的第一大优势是它非常省钱。企业不用自己花大价钱去买服务器，也不需要后续的安装、管理和维护。他们只需要每个月按照需求交一笔服务费，就能够获得所需的全部算力服务。这大大降低了企业的运营成本和运营难度，也使得他们再也离不开云计算服务商提供的服务了。

云计算第二大让企业离不开的优势是它具有极强的灵活性。公司可以根据自己的情况对硬件的需求灵活地扩张和收缩，需求高的时候就多用一些算力，需求低的时候就可以省钱少用一些。别小瞧这个灵活性，强如流媒体巨头 Netflix，也因为自己服务器的灵活性不足而被迫采用了亚马逊的 AWS 云计算服务。当初 Netflix 斥巨资从 IBM 和甲骨文（Oracle）那里买来了最好的服务器和数据中心硬件，可 Netflix 毕竟是流量大户，而且他们的流量非常不稳定，大家喜欢赶在同一时间看电影，高峰时流量非常大。为了应对流量峰值，Netflix 不断烧钱去提升自己的硬件水平，但还是无能为力。同时，他们还需要面对非高峰时期算力极大的浪费。最终，Netflix 在 2007 年找到 AWS 进行合作，将服务器全部转向了 AWS 的线上云。AWS 强大的算力和按需付费的商业模式，一举解决了 Netflix 峰值算力和算力浪费这两大难题。现在 Netflix 要想离开 AWS，几乎是一件不可能的事情。

云计算第三大让企业无法拒绝的优势，可能很多人想不到，是它的安全性。公司如果自己配置硬件，还需要聘请专业的安全团队来防止数据的泄露和损失。但现在，云服务商可以根据你的安全等级提供上不封顶的安全保障。这对于公司规避风险、提高运营效率来说都有重大意义。最经典的一次案例是美国信用记录公司 Equifax 的黑客入侵事件。2017 年，Equifax 曾被黑客入侵，大量关于个人信息的敏感数据被盗取，这起事件对于公司的影响巨大。对于信用记录公司来说，信息安全至关重要，但这又不是他们自己能够解决的。于是在被黑客入侵之后，管理层将所有的资料全部转移到了谷歌云（Google Cloud）上，希望让谷歌为他们的数据安全护航。而谷歌也为 Equifax 设置了最严格的安全认证措施。自此，Equifax 再未出现严重的网络安全事件。可见，像 Equifax 这样的公司都不得不用云服务商来保证它的安全性，更多有类似需求的公司也肯定都离不开云服务商提供的安全保障。

正是因为云计算这些让企业无法拒绝的优势，使得企业们一旦用上云服务就再也离不开它了，这就是云计算极强防御性的由来。也是因为这些优势，使得还没用上云服务的企业们也在纷纷转移至云端。理论上，只要一家公司需要网络服务、会产生数据，最终就必将成为云计算的客户，而这就是云计算确定性的由来。最终我们会意识到，在目前这个任何生意都在向线上转移的时代，在这个 AI、大数据乃至元宇宙（Metaverse）急速发展的时代，作为基础设施的云计算的需求只会越来越高。而现在这个过程还仅仅处于初级阶段，这就是云计算增长性的根源。

根据第三方数据测算，云计算的市场规模在未来五年的平均增长能够达到 16.3%，五年后就会接近万亿美元的规模。而这里说的还是包括上下游在内的整个行业，如果我们只看最熟悉的、像 AWS 这种直接对接企业的云服务，未来的平均增速甚至会高达 30% 以上。

这样一个兼具防御性、确定性和增长性的行业，自然也是兵家必争之地。好东西大家都想要，但是最终谁能够站稳脚跟，才是投资者最关心的问题。关于这个问题，有一个特别好的借鉴。云计算行业现在的格局，其实与 30 年前手机在全球开始普及时的电信行业非常类似。想象一下，如果你现在是 1990 年，你能够看到什么趋势？越来越多的人会开始使用手机，用上手机的人就很难再离开它了，而且每个人对手机的需求只会越来越大。所以，当时电信服务的需求也肯定会越来越高，行业的发展也会非常快。这其实就和现在云计算行业的商业逻辑如出一辙。而且它们还有一个共同点，那就是行业的门槛都特别高。电信服务前期要搭建信号塔等基础设施，这部分成本非常高，技术难度也很大，肯定不是谁都能进来分一杯羹的。而云计算也同样如此，前期搭建服务器、数据中心的成本也非常高，技术难度甚至更大。

30 年后的现在，我们再来看电信行业的发展轨迹，会有一种开了天眼的感觉，似乎发生在电信行业上的所有事情，也将逐一发生在云计算行业身上。首先我们能看到，电信服务商不管在哪个国家都是寡头垄断的格局。在美国，最大的三家电信公司 AT&T、T-Mobile 和 Verizon 占了大部分的市场份额，同样的情况也存在于中国、加拿大等国家。另外，如果你去看 90 年代 AT&T 的股价，就会发现它的增长速度非常迅猛，十年间翻了 15 倍还多，而这还是在它被美国政府强制拆分后的结果。当然，这里面除了手机，还有互联网普及的因素，但我们能够看到的是，作为行业龙头的 AT&T 随着行业一起快速增长的结果。

反观云计算行业，我们也能​​够合理地推测，在当前这个行业发展初期的阶段，未来云计算行业也将会迎来很长一段时间的高速增长。而在这个过程中，该行业也大概率会形成一个寡头垄断的局面。最终，也许再过十年，这些巨头将会和行业一起逐渐发展成熟，占据整个市场的垄断地位。

介绍完云计算行业，接下来看看行业内几个重要的玩家。我们先来看一下现在这个行业大致的格局。云计算行业过去三年几个重要玩家的市场份额以及市场份额的变化显示，前三家公司亚马逊、微软、谷歌就占了整个行业三分之二的市场份额：亚马逊占 33%，微软占 21%，谷歌占到 10%。从变化趋势上我们可以看出，亚马逊 AWS 的市场份额非常稳定，且领先地位也非常明显。而后两名微软和谷歌近两年的市场份额都有明显增加，尤其是微软表现得非常抢眼。但蛋糕就这么大，他们抢走的是谁的份额呢？很明显就是所谓的“其他公司”。这说明资源正在不断地向大公司集中，寡头垄断的格局也在不断地加固。

顺带一提，第四名的阿里巴巴是一个比较特殊的存在，它的市场几乎都在中国，跟其他几家公司没有太强的直接竞争，而中国国内的云计算行业又有自己的竞争格局，我们这里就不做过多分析了。第五名的 IBM 虽然还有 3% 的市场份额，但是下降的趋势也很明显，估计过不了两年就会被划归为“其他玩家”了，我们这里也不做分析。所以，我们接下来的重点就是分析亚马逊、微软和谷歌各自的云计算业务发展。

我们先来看行业龙头，亚马逊的 AWS (Amazon Web Services)。现在的 AWS 最类似于当年的 AT&T。作为行业龙头，它全程享受了行业高速增长的红利。AWS 能够坐拥今天的位置，跟它的先发优势有莫大的关系。整个云计算的商业模式就是由亚马逊最先发明出来的，它在发展的前七年里几乎没有任何竞争对手，这给了 AWS 不同寻常的先发优势。更为难得的是，随着体量的增加，它的增速和行业地位并没有出现自然的下滑，而是始终保持得非常坚挺。正因为它做得早，所以 AWS 的云服务也是行业内最为全面和成熟的。AWS 有着完整的配套服务和完善的生态系统，在细分领域中，AWS 在中小型企业、电商和国际市场中优势最为明显。它还有一点容易被忽略的优势是，对于程序员来说，AWS 可能是最为友好的，它的资料库和用户社区比起其他服务商来说要更为活跃，也更为丰富。

从数据上看，AWS 的表现少有的让人安心。最新一季的财报显示，AWS 的营收同比增长依然高达 37%，这甚至比去年同期的 32% 还要快。很显然，AWS 并没有像多数科技股那样，在后疫情时代增速放缓，反而是在继续加速增长。并且分析师预计未来三年的平均增速仍然能够高达 30%，这也符合我们对于这个行业快速发展的一个预期。不过，作为亚马逊的业务，AWS 不可避免地会受到母公司的影响。其中，投资者最担心的就是它的盈利能力。熟悉亚马逊的观众都知道，亚马逊最著名的商业逻辑就是不计成本地烧钱，它的电商业务到现在真实利润率都依然是负的。AWS 同样也是如此，为了保持高速增长和领先地位，AWS 在烧钱这一点上可以说是所有云服务商中最不手软的。但令人欣慰的是，上季度 AWS 的运营利润率达到了 35.3%，这是它的历史最高值。这可并不是 AWS 烧钱烧得少了，数据显示，AWS 的运营支出仍然增长了 37%。之所以实现了利润的大幅增加，是因为它们的规模效应开始逐渐显现。这也说明 AWS 已经开始回收云计算行业的红利，这对于投资者来说绝对是一件好事。

接下来，我们再来看行业老二，微软的 Azure。单从硬实力讲，微软理应比亚马逊更能胜任云计算的发展，奈何贝索斯更有魄力，提前了七年就开始布局。但微软也不是吃素的，仗着自己在办公软件上的优势，以及和大型企业们良好的长期合作关系，微软成功在大企业的云服务这个细分领域中占据了优势。正因为微软的客户多为大企业，也使得他们在业务稳定性上要更强一些。Azure 的另一大优势是它的网络安全。虽然这几家公司的网络安全都做得不错，但微软绝对是最强的一个。很多大公司也是冲着这一点选择跟微软合作，这未来也将会成为微软的一个主要竞争优势。

在定量上，微软对于 Azure 的信息披露非常吝啬，这也给我分析它的客观数据增加了不少难度。从仅有的数据中，我们至少可以看出一点，那就是微软 Azure 的增速是所有云计算公司中最快的。本季度 Azure 的同比增速高达 48%，比起去年同期的 50% 稍微慢了一点，但也足够令人满意。自疫情以来，Azure 基本上都能够稳定在 50% 左右的增长，它没有爆发性的增加，也不曾有过任何的下降，可见 Azure 增长的稳定性，这点和它的“亲爹”微软如出一辙。正因为微软 Azure 超高的增速以及无与伦比的稳定性，不少人都相信 Azure 有机会在几年后超越亚马逊 AWS，成为全球第一大云服务商。从之前的市场份额变化中我们也可以看出，Azure 的上升势头非常迅猛，大有超越 AWS 之势。不过，我个人倒不这么认为。AWS 也并不是原地不动，而是在随着行业一起快速增长。Azure 现在可能还在享受着新人的优势，而 AWS 已经开始回收行业的红利了。这两者的发展阶段有所不同，也不能同日而语。

最后，我们说说谷歌云（Google Cloud）。不得不说，Google Cloud 这个名字起得就有点随意。人家微软和亚马逊好歹给自己的“儿子”起个名，谷歌就直接叫“儿子”。实际上，你看谷歌云的发展，就像谷歌起名字一样，不是那么走心。本季度，谷歌云的营收增长为 43%，不如 Azure 的 48%。它 2021 年全年的增长为 47%，虽然也看得过去，但依然比不上 Azure 的 50%。这个数据其实是比较令人失望的，毕竟 AWS 在更大的体量下，依然保持着非常强劲的增长。现在谷歌云的体量不及 Azure 一半，增速还要更慢一些，这说明两者的差距会越拉越大。不过，要说谷歌跑得慢，其实一点也不冤，我甚至认为它是有意而为之。用谷歌全年的营收减去它的运营成本后发现，谷歌去年的运营支出仅比 2020 年增长了 20%，远不如 AWS 的 37%。Azure 虽然没有披露相关数据，但从其他数据中我们也可以推断出，其投入的增长也会远超 20%。谷歌又想让马儿跑，又想让马儿不吃草，这怎么可能呢？谷歌云的掉队也就不难理解了。

如果我们换个角度，来看云计算对各家公司的重要性，很多问题可能就迎刃而解了。很显然，AWS 对于亚马逊来说是极为重要的。AWS 的市值占了亚马逊的一半还多，亚马逊的盈利几乎全部来自于 AWS。而 Azure 对于微软来说虽然也很重要，是微软最重要的增长引擎，但毕竟微软家大业大，还有很多其他业务要照顾，云计算的重要性不如 AWS 之于亚马逊那么高。至于谷歌就更别说了，云计算的收入仅占总收入的不到 10%，在公司的未来战略中，AI 的地位要远高于云计算。所以，他们没那么上心，也​​就能够理解了。这些话 CEO 肯定是不会告诉你的，只能靠我们投资者自己体会。

关于云计算行业，今天就跟大家分析这么多。这期视频我是希望能够做到尽量客观，所以抹去了很多我自己的观点，目的就是为了不妨碍各位观众独立思考。在如今这种动荡的市场环境下，冷静的思考绝对是第一要务。
