# 什么是“1月效应”？

什么是“1月效应”（January Effect）？我先给大家看一张图，这是全球主要金融市场平均每个月的收益表现。可以看到，1月份是所有月份中表现最强的，平均回报率能达到2.86%，远高于剩下11个月的表现。

“1月效应”远不止如此。你当然可以直接买大盘来乘上这波趋势，但这会浪费“1月效应”中最有价值的机会，因为“1月效应”中势能最强、最具有爆发力的是一类非常特殊的股票。这类股票每每在1月爆发，而且这个规律持续了近50年，从未改变过。而现如今，种种迹象表明，今年的“1月效应”将会非常明显。即便你不去靠它赚钱，了解它也能帮你更好地看懂股市的表现。

那么，究竟什么是“1月效应”？它靠不靠谱？“1月效应”中的爆发股票都有怎样的特点？今年会有哪些股票最有可能在“1月效应”中爆发呢？本期视频，我将一一给出答案。

# “1月效应”背后的理论依据

为什么1月份市场整体的表现会是全年最好的？这是历史的巧合还是有理论依据？对于这个问题，业内做过不少深入的研究，但到目前为止都还没有一个定论。不过，“1月效应”确实有理论依据，总结下来无外乎两点核心原因。第一个源于资金面。每年的年底和来年初，往往是各大公司发奖金的时刻，很多税收优惠账户也会在1月份提高额度。很多散户会习惯性地在此时将奖金和存款一起投入到股市当中。各大基金经理们也常常在这段时间内收到不少新的资金，并且1月份也是他们放假归来、开始布局全年规划的时刻。无论是机构还是散户，1月份都是新资金入市最为集中的月份，能够带动股市的上涨。第二个被公认的原因是投资者的积极情绪。这点可以从一个非投资类的数据——“消费者信心指数”（Consumer Confidence Index）中得到佐证。“消费者信心指数”是纯粹反映情绪的数据，历史上该数据在1月份常常会出现异常的提升，也是全年平均值最高的时候。类似的，消费的信心足，放到投资上也是如此。每年1月份往往是投资者情绪最为高涨的时候，有超过三分之一的美国人会在年初制定新的投资计划，更加积极地寻找新的投资机会，导致市场上的交易动能提升，从而更容易推动股市上涨。

# “1月效应”在美股大盘上已失效？

既然1月份有如此多的上涨动能，那是不是无脑买大盘就行了呢？并非如此简单。一张来自彭博（Bloomberg）的研究图表显示，在1993年之前，标普500（S&P 500）在1月份的表现异常突出，平均收益率达到1.85%，冠绝所有月份。这和我们开头展示的全球主要经济体的股市表现完全一致。但是，到了1993年之后的30年中，美股大盘的1月份表现就变得异常平庸了，平均只有0.28%，放到全年连前六都排不进去。很显然，如果再无脑投大盘去押注“1月效应”，效果恐怕不一定会好。

# 真正的“1月效应”：输家的反弹

既然“1月效应”在美股大盘上已经如此弱，现在了解它还有用吗？有用，因为“1月效应”的底层逻辑没有变化，它依然存在于当下的美股当中，只是对有些股票是积极的，对有些股票则反而是消极的。那些真正受益于“1月效应”的股票正在变得更加集中，也变得更加隐秘。这里有一张来自《金融学期刊》（The Journal of Finance）一篇论文的图表，其中作者做出了两个投资组合。第一个投资组合叫做“输家组合”（Loser Portfolio），它是在一定体量的个股中，筛选出去年表现最差的35只股票组成的组合。这个组合是每年进行动态调整的。另一个投资组合叫做“赢家组合”（Winner Portfolio），它则是在相同的池子中筛选出去年表现最好的35只股票的组合，这个组合也是每年进行动态调整的。作者将1933年到1980年这近50年的数据综合起来，做出了这两个组合在36个月内的平均表现。整体来看，前一年的“输家组合”在未来三年时间内，整体是跑赢“赢家组合”的。重点是，对于“输家组合”来说，每年它都会出现一次明显的跳涨，而这次跳涨全部都发生在1月份。而同一时间，“赢家组合”则完全看不到这个现象，甚至在1月还不约而同地出现了下跌。

所以，“1月效应”并非是均匀发生在所有股票身上的，而是更容易发生在那些前一年表现不好的股票身上。而对于那些前一年表现不俗的股票来说，“1月效应”则表现得并不明显，甚至还有可能是副作用。这也是为什么无脑布局“1月效应”，结果恐怕不会太好的原因。

不过，这篇论文所用的数据毕竟还有些久远，那么到了现在，这个规律还存在吗？这个规律不仅存在，而且变得更强了。就在上个月，瑞银（UBS）发表的一份研报指出，如果把过去35年里标普500跌幅超过10%的股票单独拿出来看，它们在随后的1月份平均可以获得2.3%的回报率，这要大幅跑赢大盘同期的0.5%。美国银行研究部（BofA Research）最近的研究也发现，如果把06年到24年每年表现最差的100只股票单拿出来看，这些股票从12月下旬开始一直到1月底，表现一直都很好，平均能够跑赢大盘2.5%。类似的还有很多。结论非常清晰，那就是“1月效应”到今天依然存在，而且普遍存在于那些前一年表现较差的股票当中。

# 为什么只有“输家”才有“1月效应”？

为什么只有“输家”（Loser）才有“1月效应”？其中一个原因是“再平衡”（Rebalancing）。每年的年初，很多大资金都需要卖掉去年那些涨太多的股票，以防它们在投资组合（Portfolio）中占比过高，然后这些资金会再投入到持仓中那些涨得不好的股票当中，这个操作就叫做“再平衡”。由于被动资金越来越多，“再平衡”的力量也变得越来越强，这进一步放大了“输家”的“1月效应”。第二个原因则和基金经理的交易习惯有关。在美国，绝大部分基金每年都需要准备一份年报给客户，而一般的截止日期正是每年的年底。所以，一部分基金经理为了粉饰年报，就很喜欢在年底时卖掉自己的“输家”仓位，这样该仓位就不会显示在他们的年报上。而到了来年的1月份，他们又会把股票再买回来，继续之前的投资计划。

不过，这两种原因其实都还不是最重要的，最重要的原因跟税务有关。本杰明·富兰克林曾说：“这个世界上，除了死亡和税收，没有什么是可以确定的。”而正是因为这个确定性的税务原因，给美股带来了持续近百年未变的“1月效应”。在2002年，由Starks等三人研究的论文 *《Tax-loss Selling and the January Effect》* 当中，他们就专门研究了税务和“1月效应”之间的关系。在论文中，作者设计了一款十分巧妙的实验。他们为了最大化税务在收益率上的效果，没有直接拿股票进行测试，而是找出了一种税务效应最为显著的金融产品：“市政债券基金”（Municipal Bond Funds）。这种基金由于重点就在于它有一定的税务优惠，所以它的投资者基本都是对于税务最为敏感的一群人。下图就是这类基金每年的按月表现。可以看到，1月份它的平均回报率高达2.2%，这对于一个债券基金来说已经算是非常高了，而且要远高于其他11个月的表现。很显然，“1月效应”在这类基金中展现出了极为明显的效果。可见，税务原因在“1月效应”中起到了十分重要的作用，而这种作用随着时间的推移，不仅没有减弱，还在不断地加强。

就在今年，Carrier等人发表的论文 *《Tax-loss Selling and the January Effect Revisited》* 中，他们就利用了同样的方法，重新观察了2000年之后“市政债券基金”的表现。结果他们发现，1月份这类基金实际的回报率，从2000年之前的2.36%增长到了2000年之后的3.05%，甚至到了2007年之后飙升到了3.4%。这说明在20年前税务效应被证明之后，到今天，税务效应不仅没有消解，反而在不断地增强。

为什么税务原因会导致“1月效应”？答案很简单，就是为了避税。投资者为了避税，往往会在前一年卖掉自己的“输家”来抵扣投资收益，然后到了第二年年初，他们会再把“输家”的股票买回来重新布局，有些人甚至会加大力度，试图早日解套。所以到了1月份，很多新的资金就会冲进这些去年亏损的股票里，这就促进了这些股票在这段时间的表现。而且这些年来，随着越来越多的散户入市，再加上美国的税法愈发严格，税务效应的影响也变得越来越大。往后看，“1月效应”在“输家”身上的体现，恐怕还会比论文中回测的效果还要更为强烈。

# 2025年或是最强“1月效应”年

而就2025年实际的市场情况来看，我们很可能即将迎来一个最为强烈的“1月效应”。原因之一是“再平衡”。从数据上看，2025年的“再平衡”资金很可能是史无前例的。图表展示的是过去70年来，年末这周美股的表现。可以看到，2024年的表现是有史以来最为糟糕的。2024年年底，美股曾连续四个交易日出现下跌，这是58年来最长的年末连跌记录。而实际上，这些天市场并没有出现什么新的负面消息，这种不寻常的下跌就很可能跟“再平衡”有关。因为过去两年赚钱的股票很多，而且赚钱的股票市值普遍还都很大，“再平衡”的盈利资金一旦撤出，就是巨大的体量。而一旦这些资金再投入回“输家”当中，对“输家”股票的表现将会是一个不小的促进。从税务效应上看，结论也是类似的。过去两年美股的表现实在太好了，这两年只要参与了市场的投资者基本都是赚钱的，这使得避税成为了很多投资者不得不考虑的问题。所以很多人在去年年底时就卖掉了手中亏损的股票，那么到了今年年初，他们很可能还得找机会再把它们买回来。而在这两年的牛市行情下，还能够保持亏损的股票其实并不多，如果去年年底大家抢着卖掉这些股票来抵税的话，那么到了今年1月，对这些股票的再投资就很可能给它们带来不错的回报。

# 风险提示与总结

注意，这里不是推荐股票，这些股票的基本面如何，这里没有分析，只是单纯从“1月效应”的角度出发，为大家筛选出了可能的标的。

最后，必须强调一下风险。首先，“1月效应”说到底是一种历史规律，而凡是规律就一定会有例外。我们前面讲的所有结论，只是在概率上更可能发生而已，并不能作为我们投资的唯一依据。另外，“1月效应”是一个非常短期的趋势。根据沃尔夫研究公司（Wolfe Research）的研究，他们发现那些在“1月效应”中表现不俗的股票，通常也会在2、3月份出现一定的股价回归，而且它们最终全年的表现也不会太好。所以综合来看，如果真想布局“1月效应”，那只能把它当成是一种投机操作，而且是那种极短期的投机操作。这就意味着必须要控制好风险。比如，千万别想着因为“1月效应”就可以全仓（All in）某只股票。就是投机，也要用小资金来尝试。赚钱了就当是给过年加几个菜，亏钱了也不要留恋，果断止损。要知道，这些“输家”之所以表现不好都是有原因的，长期持有“输家”的风险显然会更高。

了解“1月效应”也不一定非要用它去赚钱。我认为，了解“1月效应”最重要的作用是帮助我们看懂美股涨跌背后的逻辑，从而更加安心地投资。比如，你看到了“输家”的股价出现了反弹，就能知道，这也许并非是基本面的改变，而仅仅是“1月效应”在作祟。再比如，你看到好股票涨势承压，也不一定是他真有问题，而不过是“再平衡”所带来的短期影响。抑或是，单纯是因为了解了“1月效应”，你能够对于保持投资更有信心。无论是哪一点，都能够帮助你更加安心地开启你的2025年投资。
