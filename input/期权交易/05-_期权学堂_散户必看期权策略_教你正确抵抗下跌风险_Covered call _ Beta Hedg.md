### **第一种策略：抛补买权 (Covered Call)**

必须吐槽一下这个中文翻译，感觉起名的人就是故意不想让你用这个策略赚钱，光是“抛补买”仨字就拐了四道弯。其实英文相对好理解，Covered Call 和上期提到的 Naked Put Selling 相对应。Naked Put Selling 是裸卖看跌期权 (put option)，也就是你不持有股票，单纯卖 put；而 Covered Call 则与之相反，是你持有股票的同时，再去卖一个看涨期权 (call option)，因而叫 Covered Call。

我们先简单解释一下这个策略的逻辑。这个策略分成两部分：一个是持有股票，一个是卖这只股票的 call option。我们还从上期讲过的通俗逻辑出发，持有股票很简单，你肯定是看涨股票。另一方面，上期提到买 call 就是看涨股票，那么卖 call 也很好理解，就是与之相反，看跌股票。也就是说一个策略里，你既看涨了股票，也看跌了股票，那怎么赚钱呢？

这就要提到期权 (option) 的具体性质了。期权由于行权价格的存在，它的看涨或看跌和单纯持有股票或做空股票的看涨和看跌是不一样的。比如单纯持有股票，本质上你是说这只股票不管在哪个价格你都认为它要涨，且股票在任何价格是上涨你都能赚钱。而期权不同，虽然买 call 也是看涨股票，但只有当股价高于行权价时你才能赚钱。说白了，你是在行权价之上看涨。相反，卖 call 也就是看跌，你也是在行权价之上看跌，也就是股价在行权价之上越涨，你越亏钱。换一个角度来讲，你之所以卖 call，是因为你认为股票在到期前不会涨过行权价，这时你就可以通过卖 call 赚取一点权利金。

我们将持有股票和卖 call option 相结合会发现，在 call option 到期之前的这段时间里，如果股价低于行权价，那么你就和持有股票的效果相同，股票涨多少你就能赚多少。如果股价涨超行权价，你就不能继续再赚钱了，股价不管涨到多高，你最多就赚到行权价格为止。这个策略原理看似复杂，其实通俗的逻辑就是，你牺牲掉股票一部分未来的上涨潜力，来换取当前能够赚取的一部分收入，而这个收入就是通过卖 call 挣得的权利金。

为了更好地解释 Covered Call 这个策略，我还是给大家举几个我自己的投资案例。Covered Call 策略可应用的一类典型的股票是那些长期持有并且相对稳定的股票。在我的实仓里，麦当劳就是典型的这类股票。麦当劳是我持有时间最长的一只股票，麦当劳的 Covered Call 策略也给我带来了长期稳定的收益。

- **案例一：麦当劳**
  例如今年6月19日，麦当劳股价186美元，此时我卖出一张麦当劳的看涨期权合约 (call contract)，总计100个 call option，一周后到期，行权价190美元，总共获得了220美元的权利金。之所以做这个操作，是因为麦当劳是一个股价浮动相对较小的公司，我预计麦当劳在一周内股价不会涨超190美元。即便涨超190美元，我也愿意牺牲超出的部分来换取这220美元的权利金。最终麦当劳股价一周后从186美元下跌至180美元，这个220美元的权利金我就算拿稳了。虽然股价有所下跌，但是这220美元的权利金也为我弥补了一部分损失。

可能有人会觉得这220美元不算什么，我给大家算一笔账。一周220美元的收益，相当于我持有100股麦当劳股票总价值的1.18%。如果每周都能成功交易 Covered Call 策略，那么一年的收益将是61%。当然，谁也不会每次都判断正确，如果你仅仅判断正确一半，也有每年30%的收益，也是相当可观了。值得注意的是，Covered Call 策略本身是不会让你亏钱的，即便你判断失误，股价涨超行权价，你所承担的风险也仅仅是少赚了超出部分那笔钱而已。

另一个利用 Covered Call 策略的思路是，用卖 call 赚得的钱来弥补可能到来的损失。这种方式适用于你打算入手一只不确定性很强的股票时。

- **案例二：波音**
  比如今年6月11日，美股出现3月以来最严重的一次单日下跌，其中波音公司单日跌幅竟达到了16.5%。由于之前视频对波音做过分析，其公司基本面还算良好，此时我认为是一个很好的入场时机。然而市场不确定性很高，波音股票又长期处于风口浪尖的地位，也不无继续下跌的可能性。于是此时我以170美元的价格买入波音股票的同时，又卖出了一张行权价在220美元、两个月后到期的 call option，价格为13美元。13美元的 call option 价格相当于7.6%的股票费用，也就是说能够给我弥补潜在7.6%的下跌风险。如果波音两个月后涨超220美元，那么我就要以220美元的价格卖出波音股票，相当于我变相地给自己设定了一个止盈价格。然而就算这种情况发生，两个月的时间，波音股票能为我带来30%的股票收益，外加7.6%的期权收益，我也是相当满意了。

可见，Covered Call 策略既可以给我们提供相对稳定的收入，也可以用来降低一定的风险。

### **第二种策略：贝塔对冲 (Beta Hedging)**

然而说到降低风险，没有哪一种策略能够做到我们接下来要讲的这种期权策略同样的抗风险效果，这就是贝塔对冲 (Beta Hedging)。这种策略被对冲基金们广泛使用，是一种可以彻底抵消系统性风险的策略。说白了，就是不管大盘涨还是跌，对于你投资组合都没有任何影响。你的收益完全取决于你所买公司的好坏，也就是说如果你买的公司表现超过平均值，你就能赚钱；反之你虽然会亏，但也仅仅是承担低于大盘的那一部分亏损。这是一个很考验你投资判断力和对已有投资信心的策略，也是一个风险极低的策略。

这种策略适合相对保守的投资者，尤其适用于当前的美股市场。2020年二季度的美股市场，有着过去20多年以来最强劲的上涨表现，标普500指数 (S&P 500) 涨超500点，涨幅高达20%。如果你成功搭上了这波顺风车，相信你如今的收益表现也一定十分亮眼。如果此时你想固定住已有收益，但又不想卖掉手中的股票，因而失去潜在的机会，或者错过下一次的入场时机，那么 Beta Hedging 就是一个十分适合你的策略。

具体的操作其实也并不复杂。最简单的办法是，你可以买入与自己已有投资比例相同的大盘反向指数，例如 **SQQQ**。也就是说做空等比例的大盘，来实现完美对冲系统性风险的目的。然而通过买反向指数做空大盘有一个很大的问题，那就是你需要有与你现有投资等量的资金来做空大盘。即便你买三倍指数做空，依然是个不小的数目。

这时，期权组合就能派上用场了。上期提到，期权 (option) 可以理解成加了杠杆的股票，而且是加了10倍、20倍的杠杆。所以从原理上，对冲基金们惯用的一个办法是，通过买指数基金的 put，同时卖出相同行权价格的 call，来实现做空大盘的目的。这背后的原理相对有些复杂，主要应用的规则是看跌-看涨期权平价关系 (Put-Call Parity)。简单说就是，股票、put option 和 call option 这三者之间存在着一种稳定的等式关系，三者中任意一个都可以被另外两个通过一定的组合方式表达出同样的效果。而做空股票就可以被“买 put、同时卖 call”这样的组合实现同样的效果。它的优点是可以花小钱办大事。

举个例子，假设你现在持有价值10万美元的股票，你可以通过操作标普500指数 (S&P 500) 的指数基金 **SPY** 的期权来实现做空大盘的目的。SPY 当前的价格是314美元一股，想要完全实现10万美元等比例的对冲效果，大概需要318股。因为期权合约 (option contract) 都是以100个期权为单位的，所以我们可以粗略地用三个期权合约，也就是300个期权来实现差不多的对冲效果。

此时，一个月到期、行权价在314美元的 put option 价格为7.75美元，买入300个 put option 的成本为2325美元。同时，一个月到期、行权价同样在314美元的 call option 价格为7.5美元，卖出300个 call option 能够得到2250美元。将这二者结合，就可以做到做空10万美元大盘的效果。所以你看，你仅仅用了75美元的价格，就成功在一个月内对冲了价值10万美元的投资，比起花几万美元去买反向指数要划算太多了。

我在这里提 Beta Hedging 这个策略，一个原因是因为它的底层逻辑很直观，容易理解，操作也简单实用。你随时想要给自己放个假，不想承担过多的风险，或者想要锁定 (lock in) 收益时，都可以拿出来用用。另一个原因是想给大家科普一下对冲基金的策略。其实对冲基金也没有我们想象的那么高大上，虽然他们的一些策略是我们普通投资者很难做到的，例如量化交易、无风险套利等，但是也有很多像 Beta Hedging 这样实用的策略，是我们普通投资者可以借鉴的。我刚好有对冲基金这方面的资源，但不知道大家是否感兴趣。如果你想了解更多的适用于我们普通投资者的对冲基金策略，请在评论区给我留言。如果想看的人多，我就做几期试试水。

这期介绍的期权策略主要还是一些需要判断涨跌的策略，即方向性策略 (Directional Strategy)。还有很多有趣的期权策略来不及讲，比如股票涨也赚钱、跌也赚钱的策略，如勒式策略 (Strangle) 和跨式策略 (Straddle)；再比如股票长期横盘不动，反而能赚钱的策略，如铁鹰策略 (Iron Condor) 和蝶式策略 (Butterfly)。还有很多特殊时机的交易策略也很有意思，比如公司发财报要怎么交易，特朗普发推特 (Twitter) 要怎么赚钱等等。期权策略很有趣，玩好了也很赚钱。如果我能做到，那么你们肯定也能做到。
